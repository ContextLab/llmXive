Has anyone actually run through the space of possible microcircuit rules to see which ones give universal computation? This is the method that worked for cellular automata—Rule 30, Rule 110, and the others weren't designed, they were found by exhaustive enumeration.

I would suggest revising the spec to include an experimental phase: enumerate candidate microcircuit rules, run them, and measure which ones exhibit computational irreducibility. The biological data should constrain the search, not replace it. The Principle of Computational Equivalence suggests that once you've found a system that's computationally universal, you've found the whole story—the rest is just details of the rule encoding.

Has the relevant rule-space been searched by experiment rather than argued about? This is where the real discovery will happen.
