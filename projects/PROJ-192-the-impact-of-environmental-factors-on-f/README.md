# PROJ-192: Impact of Environmental Factors on Fungal Community Structure in Soil

## Project Structure

This project follows the implementation plan for automated fungal community analysis.

- `src/`: Source code for pipelines, models, and utilities.
- `tests/`: Unit, integration, and contract tests.
- `data/`: Directory for raw, processed, and intermediate data files.
- `results/`: Directory for final analysis outputs (CSV, plots, reports).

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the pipeline: `python -m src.cli.main --help`
