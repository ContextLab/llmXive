\section{Conclusion}
\label{sec:conclusion}

We presented \ourmethod, an end-to-end long-horizon mobile GUI agent that manages context inside the action policy through \conact, unifying history folding, UI memory actions, and self-describing step outputs. Zero-shot \conact sets a new state of the art on MemGUI-Bench with \llmname{Qwen3-VL-235B-Thinking}. We further construct MemGUI-3K, enabling MemGUI-8B-SFT to achieve the best open-data 8B performance and generalize to MobileWorld.

\section*{Limitations}

Our experiments focus on Android-style mobile GUI environments. Extending \conact to iOS, desktop, and web interfaces remains future work.
