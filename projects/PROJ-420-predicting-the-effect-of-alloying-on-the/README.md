# Predicting the Effect of Alloying on the Poisson's Ratio of Aluminum Alloys

An automated science pipeline to predict Poisson's ratio in aluminum alloys using compositional data and machine learning.

## Project Structure

- `code/`: Source code for data extraction, cleaning, modeling, and analysis
- `data/`: Raw and processed datasets
- `models/`: Trained model artifacts
- `docs/`: Documentation and generated reports
- `tests/`: Unit and integration tests
- `specs/`: Feature specifications and design documents
