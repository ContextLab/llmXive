# The Impact of Bounded Confidence on Opinion Polarization Speed

**Field**: psychology

## Research question

To what extent do specific network topologies (e.g., scale-free vs. small-world) disrupt the universal power-law scaling of convergence time in bounded confidence models, and what structural features of the network determine the divergence of the scaling exponent $\gamma$?

## Motivation

While foundational bounded confidence models suggest a universal relationship between confidence thresholds and convergence speed, it remains unclear whether this scaling law holds across diverse real-world network structures or breaks down due to specific topological features. Understanding these structural dependencies is critical for predicting how quickly societies might fragment into echo chambers under different interaction patterns, moving beyond idealized random graphs to more realistic social topologies.

## Related work

- [Inside the Echo Chamber: Disentangling network dynamics from polarization (2019)](https://arxiv.org/abs/1906.09076) — Establishes that network homophily and topology are distinct drivers of polarization, necessitating separate analysis of structural effects on convergence speed.
- [A Psychologically-Motivated Model of Opinion Change with Applications to American Politics (2014)](https://arxiv.org/abs/1406.7770) — Demonstrates how agent-based models can map individual behavioral rules to macro-level polarization but focuses on behavioral rules rather than network-structure sensitivity.
- [A Model of Opinion Dynamics with Bounded Confidence and Noise (2011)](https://arxiv.org/abs/1106.0008) — Introduces stochastic elements to bounded confidence models, highlighting how noise affects stability, though it does not systematically quantify the speed-threshold scaling across topologies.
- [Polarized Ukraine 2014: Opinion and Territorial Split Demonstrated with the Bounded Confidence XY Model, Parameterized by Twitter Data (2017)](https://arxiv.org/abs/1706.00419) — Validates the bounded confidence framework against empirical data but relies on a specific real-world network rather than a systematic exploration of structural universality.

## Expected results

We expect to find that the universal power-law scaling of convergence time ($T \propto (\epsilon - \epsilon_c)^{-\gamma}$) is preserved in regular and random graphs but significantly disrupted in scale-free networks, where hub structures accelerate or decelerate convergence non-uniformly. The scaling exponent $\gamma$ is hypothesized to correlate positively with network assortativity and inversely with the average path length, indicating that specific structural metrics determine the robustness of the scaling law.

## Methodology sketch

- Generate synthetic network ensembles (Erdős-Rényi, Barabási-Albert, Watts-Strogatz) with $N=500$ nodes using NetworkX to ensure reproducibility and fit within GHA memory limits.
- Implement a discrete-time Hegselmann-Krause agent-based model in Python, initializing agent opinions uniformly in $[0, 1]$.
- Systematically sweep the confidence threshold $\epsilon$ from 0.05 to 0.50 in increments of 0.05 across all network types.
- Execute 50 independent simulation runs per configuration with different random seeds to account for stochastic variance in initial conditions and update orders.
- Measure convergence time as the number of iterations until the maximum opinion change across all agents falls below $10^{-4}$.
- Calculate structural metrics (average path length, clustering coefficient, assortativity) for each generated network instance.
- Fit a power-law model $T = A(\epsilon - \epsilon_c)^{-\gamma}$ to the convergence time data for each topology and extract the scaling exponent $\gamma$.
- Perform a multiple linear regression analysis to determine if the extracted $\gamma$ values are predicted by the network structural metrics (path length, assortativity, etc.), using a distinct dataset of networks for validation to ensure independence from the fitting process.
- Visualize the results by plotting convergence time distributions and $\gamma$ vs. structural metric scatter plots using Matplotlib.
- Optimize the simulation loop and limit agent count to $N=500$ to ensure the entire pipeline completes within the 6-hour GitHub Actions runner limit.
