# Research: Simulation-Based Sensitivity Analysis of Framing Effects

## 1. Research Question & Hypothesis

**Primary Question**: How does the statistical power of a mixed-effects model vary as a function of the true framing effect size (delta) in a simulated between-subjects experiment?

**Hypotheses**:
1.  **H1 (Sensitivity)**: The probability of detecting a significant framing effect (p < 0.05) increases monotonically with the true effect size (delta).
2.  **H2 (Design Validation)**: A sample size of N=300 (150 per condition) provides ≥80% power to detect a small-to-medium effect (d=0.3) as specified in the design.
3.  **H3 (Moderation)**: The effect of framing on severity/sharing may vary by content domain (Health, Politics, Science), though the primary analysis focuses on the main effect.

*Note*: This is a **sensitivity analysis**, not a test of whether the effect exists in the real world. The "effect" is generated by the simulation; the study tests whether the *design* can detect it.

## 2. Methodology & Statistical Plan

### 2.1 Simulation Design (Between-Subjects)

Since the target dataset (MPSD-v2) does not contain experimental framing conditions, this study employs a **simulation-based sensitivity analysis** with a **between-subjects design**.

1.  **Stimulus Selection**: N=10 unique headlines are sampled from the source data (or generated synthetically if no verified source exists), stratified by `content_domain` if possible.
2.  **Participant Assignment**:
    - Total N = 300 participants.
    - Each participant is assigned to exactly **one** stimulus and **one** framing condition.
    - **Design**: 10 stimuli × 2 conditions × 15 participants per cell = 300 total responses.
    - **Participant IDs**: Unique per response (no repeated measures per participant). This justifies the exclusion of `(1 | participant_id)` in the model.
3.  **Synthetic Response Generation**:
    - **Severity Rating**: Generated from a parametric distribution (Normal, truncated to 1-7) with a mean shift for the "harm" condition.
    - **Sensitivity Protocol**: The simulation will run multiple iterations, varying the input effect size (delta) across a range of magnitudes. This maps the power curve rather than testing a single fixed effect.
    - **Sharing Intention**: Generated as a binary outcome (0/1) using a logistic function where the probability decreases for the "harm" condition.
    - **Random Seed**: Fixed (e.g., `42`) to ensure reproducibility (Constitution Principle I).

### 2.2 Statistical Models

#### Primary Analysis: Mixed-Effects Linear Model (Severity)
- **Outcome**: `severity_rating` (Continuous, 1-7).
- **Fixed Effect**: `framing_condition` (Categorical: Harm vs. Fact).
- **Random Effect**: `(1 | stimulus_id)` to account for stimulus-specific variance.
- **Equation**: $Severity_{ij} = \beta_0 + \beta_1(Framing_{ij}) + u_j + \epsilon_{ij}$
- **Correction**: Bonferroni adjustment applied to the p-value of $\beta_1$.
- **Ordinality Assumption**: While Likert scales are ordinal, 7-point scales are often treated as continuous in mixed models due to robustness. We will verify the distribution of residuals and report if skewness is severe. If severe, an ordinal model (`clmm` from `ordinal` package) will be run as a sensitivity check.

#### Secondary Analysis: Logistic Regression (Sharing)
- **Outcome**: `sharing_intention` (Binary).
- **Predictors**: `framing_condition`, `content_domain`.
- **Equation**: $\log(\frac{p}{1-p}) = \beta_0 + \beta_1(Framing) + \beta_2(Domain)$
- **Metric**: Odds Ratio (OR) for the framing effect.
- **Note**: Since the design is between-subjects (unique participant per row), no random effect for participants is needed.

#### Power Analysis
- **Tool**: `pwr` package (R).
- **Parameters**: $\alpha = 0.05$, Power $\ge 0.80$, Effect Size $d = 0.3$, N = 300.
- **Decision Rule**: If calculated power < 0.80, the plan flags a warning and proceeds with a "low power" note, not aborting the run.

### 2.3 Multiple Comparisons & Rigor
- **Family-Wise Error**: Bonferroni correction applied to the primary hypothesis tests.
- **Collinearity**: Not applicable due to experimental design, but VIF will be calculated for safety.
- **Causal Inference**: Claims are limited to the *simulated* context. The random assignment of framing in the simulation justifies causal language *within the simulation*, but the study explicitly states these are *potential* effects pending real-world validation.

  - Range of `severity_rating` (1-7).

## 5. Risk Assessment

| Risk | Mitigation |
| :--- | :--- |
| **Dataset Missing** | If no verified source with headlines exists, generate synthetic stimuli automatically. |
| **Model Convergence** | Use `lme4` with `lmerTest`; if singular fit occurs, report warning and proceed with fixed-effects only. |
| **Power < 0.80** | If power analysis fails, flag in `results.md` and proceed with a "low power" warning, not aborting the run. |
| **Collinearity** | Not applicable due to experimental design, but VIF will be calculated for safety. |
| **Ordinality Violation** | Check residual distribution; if severe skew, run `clmm` as sensitivity check. |
