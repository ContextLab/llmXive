# Project Glossary

This document defines key terms and acronyms used in the project.

- **HCP:** Human Connectome Project. A large-scale dataset containing multimodal brain imaging and behavioral data.
- **ICA-FIX:** Independent Component Analysis with FMRIB's ICA-based X-noiseifier. A method for automated artifact removal in fMRI data.
- **fMRI:** Functional Magnetic Resonance Imaging. A technique to measure brain activity by detecting changes in blood flow.
- **tSNR:** Temporal Signal-to-Noise Ratio. A measure of data quality in fMRI, calculated as the mean signal divided by the standard deviation over time.
- **FD:** Framewise Displacement. A measure of head motion during an fMRI scan.
- **Schaefer Atlas:** A high-resolution parcellation of the cerebral cortex into 400 regions (nodes) based on functional connectivity.
- **Participation Coefficient:** A graph metric quantifying how well a node is connected to other modules in the network.
- **Within-Module Degree:** A graph metric quantifying the connectivity of a node within its own module.
- **FDR:** False Discovery Rate. A statistical method for correcting multiple comparisons (e.g., Benjamini-Hochberg).
- **PCA:** Principal Component Analysis. A dimensionality reduction technique used to identify latent factors in network metrics.
- **MNI:** Montreal Neurological Institute. A standard coordinate system for brain imaging data.
- **NIfTI:** Neuroimaging Informatics Technology Initiative. A file format for neuroimaging data (`.nii`, `.nii.gz`).
- **FSL:** FMRIB Software Library. A comprehensive library of analysis tools for fMRI, MRI, and DTI brain imaging data.
- **AFNI:** Analysis of Functional NeuroImages. A software suite for analysis and display of functional neuroimaging data.
