# llmXive: DomainShuttle Follow-up

Automated research pipeline for extending "DomainShuttle: Freeform Open Domain Subject-driven Text-to-video Generation".

## Project Structure

- `src/`: Source code for data loading, models, and analysis.
- `tests/`: Unit and integration tests.
- `data/`: Directory for raw, processed, and result artifacts.
- `specs/`: Feature specifications and contracts.

## Setup

1. Create a virtual environment:
 ```bash
 python -m venv venv
 source venv/bin/activate # On Windows: venv\Scripts\activate
 ```

2. Install dependencies:
 ```bash
 pip install -r requirements.txt
 ```
