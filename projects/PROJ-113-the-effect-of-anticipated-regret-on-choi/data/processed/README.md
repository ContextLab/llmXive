# Processed Data Directory

This directory contains cleaned, transformed, and feature-engineered datasets ready for analysis.

## Schema and Structure

### Primary Output Files
1. **regret_proxy_v1.csv**
 - Generated by: `code/features.py` (calculate_min_max_regret)
 - Columns: `participant_id`, `trial_id`, `regret_proxy`, `deferral`, `option_count`, `perceived_risk`
 - Description: Main dataset for User Story 1 analysis

2. **excluded_participants.csv**
 - Generated by: `code/modeling.py` (T028)
 - Columns: `participant_id`, `reason`
 - Description: List of participants excluded from random effect estimation

## Data Integrity
- Processed data is derived deterministically from raw data
- Versioning is applied via filename suffixes (e.g., `_v1`, `_v2`)
- Do not manually edit files in this directory

## Processing Pipeline
Raw data flows: `data/raw/` → `code/ingest.py` → `data/processed/` → `code/modeling.py` → `data/results/`
