# Results Directory

This directory contains the final outputs of the analysis, including model coefficients, robustness reports, and sensitivity analysis results.

## Schema and Structure

### Primary Output Files
1. **coefficients.csv**
 - Generated by: `code/modeling.py` (T030)
 - Columns: `term`, `beta`, `std_err`, `p_value`, `odds_ratio`
 - Description: Mixed-effects model regression coefficients

2. **model_summary.txt**
 - Generated by: `code/modeling.py` (T030)
 - Description: Human-readable summary of model fit statistics

3. **robustness_report.md**
 - Generated by: `code/robustness.py` (T038)
 - Description: Comparison of coefficient directions across datasets

4. **sensitivity_analysis.csv**
 - Generated by: `code/sensitivity_analysis.py` (T039)
 - Columns: `proxy_variation`, `odds_ratio`, `confidence_interval`, `p_value`
 - Description: Sensitivity of results to different proxy definitions

5. **proxy_validity_report.md**
 - Generated by: `code/robustness.py` (T037)
 - Description: Validation of regret proxy against self-report data

## Data Integrity
- Results are generated deterministically from processed data
- Versioning is applied via filename suffixes if re-run
- Do not manually edit files in this directory
