# Figures Directory

This directory contains all generated plots and visualizations.

## Schema and Structure

### Expected Outputs
- **regret_distribution.png**: Histogram of regret_proxy values
- **deferral_rate_by_options.png**: Bar chart of deferral rates by option count
- **model_fit.png**: Diagnostic plots for the mixed-effects model
- **sensitivity_plot.png**: Visualization of sensitivity analysis results

## Naming Convention
`{plot_name}_{date}.png` or `{plot_name}_{date}.pdf`
- PNG for web/dashboards
- PDF for publication-quality figures

## Generation
Figures are generated by:
- `code/modeling.py` (diagnostic plots)
- `code/robustness.py` (comparative plots)
- `code/sensitivity_analysis.py` (sensitivity plots)
