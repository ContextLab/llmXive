# Comparative Analysis of Molecular Fingerprints for Pesticide Toxicity Prediction

## Project Structure

- `data/raw/`: Raw datasets (e.g., Tox21)
- `data/processed/`: Filtered and processed data
- `code/`: Python scripts for the pipeline
- `tests/`: Test suite

## Usage

Follow `quickstart.md` to run the full pipeline.

## Prerequisites

- Python 3.9+
- `pip install -r requirements.txt`
