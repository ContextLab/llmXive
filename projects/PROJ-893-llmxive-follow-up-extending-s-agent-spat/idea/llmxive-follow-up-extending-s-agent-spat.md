# llmXive follow-up: extending "S-Agent: Spatial Tool-Use Elicits Reasoning for Spatial Intelligence"

## Summary of the prior work
S-Agent introduces a spatial tool-use agentic paradigm that transforms spatial reasoning from isolated frame-level predictions into a continuous process of spatio-temporal evidence accumulation. By employing a VLM as a semantic planner that orchestrates a hierarchy of 2D grounding and 3D lifting tools, alongside a dual-memory system for scene and reasoning state, it enables training-free improvements in spatial intelligence for existing VLMs. The framework further demonstrates that distilling these agent-generated trajectories yields a compact, high-performing model (S-Agent-8B) capable of complex spatial tasks like counting and relative positioning.

## Proposed extension
**Research Question:** Can the "spatio-temporal evidence accumulation" paradigm of S-Agent be effectively distilled into a purely symbolic, CPU-tractable reasoning engine that replaces the VLM planner with a deterministic constraint solver, thereby achieving comparable spatial accuracy on static scenes while eliminating the computational overhead of neural inference? This matters because it tests whether the core intelligence of S-Agent lies in the neural VLM's "understanding" or strictly in the structured tool-use logic, potentially enabling spatial reasoning on edge devices without GPUs.

## Methodology sketch
We will construct a dataset of 1,000 static multi-view scenes from the S-Agent-300K benchmark, extracting the ground-truth tool-call sequences (the "reasoning traces") generated by the original S-Agent. The procedure involves replacing the VLM semantic planner with a lightweight, CPU-based constraint satisfaction problem (CSP) solver that takes the 3D geometric evidence (lifted by the existing tools) as input and uses the historical traces to learn a policy for selecting the next evidence-gathering tool. We expect the resulting symbolic agent to achieve >85% of the original S-Agent's accuracy on spatial counting and relative position tasks while reducing inference latency by two orders of magnitude on a standard CPU, proving that the reasoning capability is separable from the neural visual encoder.
