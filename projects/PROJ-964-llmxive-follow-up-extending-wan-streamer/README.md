# Project: llmXive Follow-up: Extending Wan-Streamer v0.1

**Project ID**: PROJ-964
**Status**: Active

## Description
This project implements the follow-up research to extend the "Wan-Streamer v0.1" system.
It focuses on extracting time-series latent vectors, training lightweight estimators,
and simulating hybrid inference for latency-quality trade-offs.

## Directory Structure
- `code/`: Source code for data extraction, model training, and inference.
- `data/`: Raw and processed datasets, model checkpoints, and metrics.
- `state/`: State tracking and artifact hashes.
- `docs/`: Documentation and design specs.
- `tests/`: Unit, integration, and contract tests.

## Prerequisites
- Python 3.9+
- See `code/requirements.txt` for dependencies.
