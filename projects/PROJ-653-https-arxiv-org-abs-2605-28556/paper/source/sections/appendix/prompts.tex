\section{Prompts}
\label{app:prompts}

This appendix contains the prompt templates used in each pipeline stage, reproduced verbatim from the running implementation. All prompts are domain-agnostic: the same template is used across the airline, retail, and telecom domains, with the domain environment (policy document, tool specification, schema) supplied through the templated placeholders. Placeholders of the form \texttt{\{name\}} are substituted with the corresponding runtime value described in Appendix~\ref{app:method_implementation}; the double braces \texttt{\{\{} and \texttt{\}\}} are Python format-string escapes that render as single braces in the text sent to the LLM and are otherwise passed through unchanged.
