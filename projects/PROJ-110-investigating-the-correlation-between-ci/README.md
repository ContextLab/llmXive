# PROJ-110: Investigating the Correlation Between Circadian Gene Expression and Metabolic Syndrome Risk

## Overview
This project analyzes GTEx data to determine if the expression levels of core circadian genes correlate with Metabolic Syndrome (MetS) risk factors, adhering to ATP-III criteria.

## Project Structure
- `code/`: Source code for data loading, classification, and analysis.
- `data/`: Raw and processed data artifacts.
- `tests/`: Unit and integration tests.
- `docs/`: Documentation and reports.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Ensure data availability as per `plan.md`.

## Execution
Run the main pipeline:
```bash
python code/main.py
```
