# llmXive: Predicting the Impact of Ball Milling on Particle Size Distribution

## Setup
1. Ensure Python 3.11+ is installed.
2. Install dependencies:
 ```bash
 pip install -r requirements.txt
 ```
3. Run the data setup script to initialize directory structure:
 ```bash
 python code/setup_data_dirs.py
 ```
