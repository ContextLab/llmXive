# Code Directory

This directory contains the source code, tests, and data processing logic for the project.
It is structured to separate source code (`src/`), tests (`tests/`), and data assets (`data/`).

## Directory Structure
- `src/`: Main application logic
- `tests/`: Unit and integration tests
- `data/`: Raw and processed data (raw is versioned, processed is generated)
- `results/`: Generated model outputs and plots
- `contracts/`: Data schemas and validation rules
- `.github/workflows/`: CI/CD configuration
