# PROJ-162: Quantifying the Impact of Data Resolution on Gravitational Wave Signal Detection

## Overview
This project investigates how data sampling resolution affects the detection probability of Gravitational Wave (GW) signals from Binary Black Hole (BBH) mergers.

## Project Structure
- `src/`: Source code modules
- `tests/`: Unit and integration tests
- `data/`: Raw and processed data artifacts
- `contracts/`: JSON Schema definitions for data validation
- `specs/`: Design documents and research plans

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run tests: `pytest`

## Execution
Follow the tasks in `tasks.md` to generate waveforms, inject signals, and analyze results.
