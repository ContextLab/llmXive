# Quickstart: Quantifying the Impact of Data Resolution on Gravitational Wave Signal Detection

## Prerequisites

- Python 3.10+
- `pip`
- Access to a network connection (for GWOSC data and PyPI).
- **Note**: This project runs on CPU only. No GPU drivers required.

## Installation

1. **Clone and Navigate**:
   ```bash
   git clone <repo-url>
   cd projects/PROJ-162-quantifying-the-impact-of-data-resolutio
   ```

2. **Create Virtual Environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r code/requirements.txt
   ```
   *Note: `pycbc` installation may take several minutes. Ensure you have a C compiler installed.*

## Running the Pipeline

The pipeline is orchestrated via `src/main.py`.

### 1. Generate and Down-sample Waveforms
```bash
python src/main.py --step generate --resolutions 4096 2048 1024 512 256 --mass-range 10 50 --distance-range 100 500
```
This creates waveforms in `data/processed/waveforms/`.

### 2. Inject into Noise
```bash
python src/main.py --step inject --noise-source gwosc --injections 50 --seed 42
```
This downloads noise segments and injects signals, saving results to `data/processed/injections.csv`.

### 3. Analyze and Profile
```bash
python src/main.py --step analyze --output results/summary.csv
```
This computes SNR degradation, detection probabilities, and resource usage.

### 4. Run Tests
```bash
pytest tests/
```

## Verifying Results

1. **Check Aliasing**: Open `data/processed/aliasing_check.png` (generated by `analyze`) to verify no spectral peaks above Nyquist.
2. **Check Detection Rate**: Look for `detection_probability` in `results/summary.csv`. It should decrease as resolution decreases.
3. **Check Resource Usage**: Verify `data/profiling/usage.json` shows memory < 6GB.
4. **Check Memory Abort**: If the job exceeds 6GB RAM, it will exit with code 1 and write a log to `data/profiling/memory_error.log`.

## Troubleshooting

- **`ImportError: No module named pycbc`**: Ensure you have `gcc` and `g++` installed. On Ubuntu: `sudo apt-get install build-essential`.
- **Memory Error**: Reduce the number of injections (`--injections` flag) or process in smaller batches.
- **GWOSC Timeout**: Check network connection. Retry with `--retry 3`.
