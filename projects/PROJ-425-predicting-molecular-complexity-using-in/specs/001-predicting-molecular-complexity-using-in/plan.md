## Summary

This project validates the hypothesis that information-theoretic molecular complexity (Shannon entropy of vertex degrees and Lempel-Ziv compression of canonical SMILES using `lzma`) correlates with established chemical properties (Synthetic Accessibility Score and QED). The approach involves downloading a random sample of molecules from a verified HuggingFace dataset, computing four metrics per molecule using RDKit (with `lzma` for LZ compression to ensure pure LZ77/LZMA algorithm), performing Pearson and Spearman correlation analysis with bootstrap resampling (a sufficient number of iterations) and multiple-comparison correction, and generating visualizations and a statistical report. 

**Critical Scope Note**: The original spec requested CID -5000. The verified dataset (`sagawa/pubchemm-canonicalized`) is a random sample of a large number of molecules, not the sequential CID 1-5000 range. The analysis is adjusted to test the hypothesis on a *representative random sample of PubChem* to ensure statistical validity and data availability. Findings are framed as applicable to general chemical space, with the CID 1-5000 limitation explicitly noted in the final report.

All processing is designed to run on CPU-only CI runners with strict memory and time constraints.

## Chunked Processing Strategy (FR-008 Compliance)

To ensure peak memory usage remains ≤ 4 GB (SC-006):
1.  **Download**: The dataset is streamed in chunks of a fixed size (e.g., a moderate batch size) using `datasets` library streaming.
2.  **Compute**: Metrics are calculated in batches. Intermediate results are appended to `data/processed/metrics.csv` immediately, and the batch is cleared from memory.
3.  **Analyze**: The full `metrics.csv` is loaded only if it fits in memory (expected for n=5000). If larger, analysis is performed in sliding windows or the sample size is reduced.
4.  **Bootstrap**: Bootstrap resampling is performed on the in-memory dataframe of valid molecules. If the dataframe is too large, a stratified subsample is used for the bootstrap phase (documented in report).

## Dataset Scope & Limitations

- **Original Spec Requirement**: CID 1-5000 (historical, potentially simpler molecules).
- **Available Verified Data**: `sagawa/pubchem-10m-canonicalized` (random sample of 10M molecules).
- **Adjustment**: The analysis proceeds on the random sample to ensure statistical representativeness and data availability. The hypothesis is reframed: "Does information-theoretic complexity correlate with SA/QED in a representative random sample of PubChem?"
- **Impact**: Results are generalizable to broad chemical space but may not strictly apply to the specific historical subset (CID 1-5000). This limitation is explicitly stated in the final report.
