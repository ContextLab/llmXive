# The Impact of Text Message Tone on Perceived Emotional Support

## Project Overview
This project investigates how text message tone (emoji usage, punctuation, length) affects perceived emotional support in different relationship contexts.

## Structure
- `code/`: Python scripts for data generation, cleaning, and analysis
- `data/`:
 - `raw/`: Generated stimuli and simulated ratings
 - `processed/`: Cleaned data and analysis results
 - `consent/`: Consent records (if applicable)
- `tests/`: Unit, integration, and contract tests
- `specs/`: Design documents and schemas

## Getting Started
1. Install dependencies: `pip install -r code/requirements.txt`
2. Run the pipeline: `python code/01_generate_stimuli.py`
3. View results in `data/processed/`
