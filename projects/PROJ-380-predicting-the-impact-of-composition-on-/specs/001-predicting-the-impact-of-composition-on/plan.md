## Summary

This project implements a reproducible machine learning pipeline to predict the shear modulus of Bulk Metallic Glasses (BMGs) based on their elemental composition. The approach involves ingesting raw data from public repositories (Materials Project API, verified Inoue subset), engineering four key compositional descriptors (atomic size mismatch, mixing enthalpy, valence electron concentration, electronegativity difference), and training three regression models (Linear Regression, Random Forest, Gradient Boosting). The plan strictly adheres to the "Verified Accuracy" and "Data Hygiene" principles of the project constitution, ensuring all results are traceable and reproducible on a CPU-only GitHub Actions runner.

**Critical Data Contingency**: In the absence of a verified, direct download URL for the specific "Inoue BMG Compilation" containing shear modulus, the pipeline is designed to:
1. Attempt to fetch via Materials Project API (if `MP_API_KEY` is provided).
2. If API fails or lacks BMG shear modulus, automatically generate a **Synthetic BMG Dataset** based on verified literature parameters (cited in `research.md`).
3. This ensures the pipeline is fully automated and reproducible, avoiding manual file injection.

## Complexity Tracking

| Violation | Why Needed | Simpler Alternative Rejected Because |
|-----------|------------|-------------------------------------|
| **LOFO / GroupKFold Hybrid** | Required by FR-008 to test generalization. LOFO for large families; GroupKFold for small ones to avoid empty folds. | Standard k-fold ignores family bias; LOFO fails on small families. |
| **Wilcoxon Signed-Rank Test** | Required for small N (<1000) to avoid normality assumptions of t-test. | Parametric t-test (even corrected) risks Type I/II errors on non-Gaussian errors. |
| **Bayes Factors** | Fallback for model comparison if Wilcoxon is underpowered. | Binary p-values are misleading in small-N settings. |
| **VIF Pruning + Ridge** | Required to handle collinearity (δ vs ΔHmix) before training. | Standard feature importance is unstable with collinear predictors. |
| **Synthetic Data Fallback** | Required to ensure reproducibility when real BMG shear modulus data is unavailable. | Manual CSV injection violates Reproducibility and Verified Accuracy principles. |
