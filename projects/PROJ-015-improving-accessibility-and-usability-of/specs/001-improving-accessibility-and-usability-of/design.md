# Design Document: Gene Regulation Simulation Context
# Note: This project adapts the gene regulation simulation framework for accessibility studies.

## Context
The underlying simulation engine is adapted from gene regulation models to provide a "complex system" scenario where users must manage resources under uncertainty.

## Accessibility Adaptation
- **Traditional Interface**: Standard resource management UI.
- **Explainable Interface**: Adds overlays explaining resource fluctuations based on simulated "gene" activity rules.

## Target Metrics
- Time to complete resource stabilization.
- Number of critical errors (system crash).
- System Usability Scale (SUS) scores.
- Explanation engagement time.
