# PROJ-015: Improving Accessibility and Usability of Complex Computer Systems for People with Disabilities

## Project Overview
This project implements an automated research pipeline to evaluate the impact of Explainable AI (XAI) interfaces on the usability of complex systems for people with disabilities.

## Structure
- `code/`: Source code for the simulator, analysis, and utilities.
- `data/`:
 - `raw/`: Immutable raw session data.
 - `processed/`: Cleaned data and analysis results.
- `specs/`: Feature specifications and design documents.
- `tests/`: Unit and integration tests.

## Getting Started
1. Ensure Python 3.11 is installed.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the setup script: `python code/setup/init_project.py`
4. Launch the simulator: `streamlit run code/simulator/app.py`
