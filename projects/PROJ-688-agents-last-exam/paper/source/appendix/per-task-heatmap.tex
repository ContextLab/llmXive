\subsection{Per-Task Instance Score Heatmaps}
\label{app:per-task-heatmap}

Figures~\ref{fig:heatmap-nearterm}--\ref{fig:heatmap-lastexam} show the mean score of every task instance under each evaluated agent system.
Rows are sorted by descending average score across all systems; columns are grouped by harness and sorted by descending average score within each group.
Task instance labels are colored by taxonomy domain (legend inset).
Gray cells indicate missing runs.
