\subsection{Timeout Analysis}
\label{app:timeout-analysis}

ALE evaluations use a five hour wall clock cap per run.
When a run reaches the cap, the harness stops the agent and the normal evaluator scores whatever artifacts are present in the output directory.
In the export used for Table~\ref{tab:main-results}, 2.9\% of evaluated runs reached the cap.
Runs that reached the cap have a mean score of 20.8, compared with 27.7 for runs that ended earlier.
