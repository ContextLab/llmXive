# Data Directory
This directory stores all raw and processed data files.

## Expected Contents
- `raw/`: Downloaded raw datasets (e.g., `barrier_dataset.csv`).
- `processed/`: Cleaned and transformed datasets.
- `descriptors/`: Generated molecular descriptors (e.g., `descriptors_semi.csv`).
- `models/`: Trained model artifacts (pickle files).

**Note**: Do not commit large binary files or datasets to version control. Use `.gitignore`.
