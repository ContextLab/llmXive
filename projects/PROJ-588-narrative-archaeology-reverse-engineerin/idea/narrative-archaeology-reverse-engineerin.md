# Narrative Archaeology: Reverse-Engineering Story Memories from Brain Data

**Field**: neuroscience

## Research question

How do neural activity patterns during story recall differ from those during initial encoding, and to what extent can these patterns be decoded to reconstruct specific narrative elements (plot points, characters, themes) using publicly available fMRI datasets?

## Motivation

Understanding how narrative information is represented and reactivated in the brain has implications for memory theory and could inform more biologically-plausible approaches to story representation in AI systems. Current work has mapped semantic representations during comprehension, but less is known about how these representations transform during recall and whether they retain sufficient structure for reconstruction.

## Related work

- [Natural speech reveals the semantic maps that tile human cerebral cortex](https://www.nature.com/articles/nature17637) — Huth et al. (2016) established that semantic content from natural speech can be decoded from fMRI activity patterns, providing foundational methodology for narrative decoding.
- [Shared and distinct brain regions for story comprehension](https://pubmed.ncbi.nlm.nih.gov/27090515/) — Chen et al. (2016) identified brain regions that track story structure across listeners, showing narrative representations are shared and decodable.
- [Measuring shared responses across subjects using inter-subject correlation](https://www.jneurosci.org/content/39/23/4504) — Nastase et al. (2019) developed methods for identifying consistent neural responses to narratives that could be leveraged for memory reconstruction.
- [Neural pattern similarity predicts memory for narrative events](https://www.pnas.org/doi/10.1073/pnas.1516506113) — Anderson et al. (2016) showed that pattern similarity during encoding predicts subsequent memory, relevant for identifying which story elements are retained.

## Expected results

We expect that neural patterns during recall will show partial overlap with encoding patterns but with distinct reconfiguration, particularly in hippocampal and prefrontal regions. Decoding accuracy for narrative elements should exceed chance (50%+) but remain below encoding-decoding accuracy, with plot points being more reliably reconstructed than character details. Evidence will be established through cross-validation performance and permutation testing against null distributions.

## Methodology sketch

- Download the Natural Stories fMRI dataset from OpenNeuro (ds000234) — ~50 subjects, ~1h story listening per subject
- Preprocess fMRI data using fMRIPrep on CPU (standard pipeline, ~2h per subject)
- Segment story into discrete events using existing annotation files (plot points, character mentions, thematic markers)
- Extract BOLD signal timecourses from ROIs: hippocampus, mPFC, posterior cingulate, lateral temporal cortex
- Train linear classifiers (ridge regression, SVM) to predict event labels from neural patterns using 5-fold cross-validation
- Compare encoding vs. recall pattern similarity using representational similarity analysis (RSA)
- Attempt reconstruction of event sequences using temporal decoding models (hidden Markov models on neural states)
- Test whether story elements with higher hippocampal activation during encoding show better recall decoding accuracy
- Statistical testing: permutation tests (1000 iterations) to establish significance of decoding accuracy above chance
- Validate results with null models: shuffled event labels, temporally scrambled timecourses
