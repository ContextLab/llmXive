# Data Model: Influence of Microstructural Features on Fatigue Life in Aluminum Alloys

## 1. Entity Definitions

### Specimen Record
Represents a single aluminum alloy test sample.
-   `specimen_id`: Unique identifier (string).
-   `alloy_composition`: String (e.g., "6061-T6").
-   `grain_size_μm`: Float (Equivalent diameter).
-   `secondary_phase_fraction`: Float (0.0 - 1.0).
-   `dislocation_density_proxy`: Float (Texture metric, labeled as proxy).
-   `fatigue_life_cycles`: Float (Outcome).
-   `microscopy_image_path`: String (Path to synthetic/real image).
-   `alloy_batch_id`: String (For grouped CV).
-   `validation_status`: String ("valid", "excluded", "imputed").
-   `is_synthetic`: Boolean (True if generated by the synthetic generator).

### Microstructural Feature
Represents a quantified metric.
-   `feature_name`: String (e.g., "grain_size", "texture_metric").
-   `feature_value`: Float.
-   `feature_unit`: String (e.g., "μm", "a.u.").
-   `measurement_method`: String ("direct", "proxy", "synthetic").
-   `is_proxy`: Boolean.

### Model Performance
Represents evaluation metrics.
-   `model_type`: String ("RandomForest", "GradientBoosting", "ElasticNet").
-   `r_squared`: Float.
-   `rmse`: Float.
-   `mean_absolute_error`: Float.
-   `confidence_interval_95`: Object `{lower: float, upper: float}`.
-   `cross_validation_type`: String ("Grouped 5-fold").
-   `parameter_recovery_score`: Float (Optional: measures how well model approximated generator coefficients).

## 2. Data Flow

1.  **Raw Input**: Synthetic generator or JSONL/CSV from URL.
2.  **Cleaned Data**: `data/processed/cleaned_specimens.csv` (N ≥ 150, no missing fatigue life).
3.  **Feature Matrix**: `data/processed/feature_matrix.csv` (Includes extracted image features).
4.  **Model Artifacts**: `results/metrics.json`, `results/models/`.
5.  **Statistical Output**: `results/anova_summary.csv`.

## 3. Constraints

-   **Missing Data**: If `fatigue_life_cycles` is missing, record excluded. If microstructural features missing <20%, impute (mean/median); else exclude.
-   **Image Fallback**: If image path is invalid, `grain_size` and `texture_metric` derived from tabular data only (simulated).
-   **Proxy Labeling**: `dislocation_density_proxy` must always be flagged as `is_proxy: true` in metadata.
-   **Synthetic Flag**: All records generated by the synthetic generator must have `is_synthetic: true`.
