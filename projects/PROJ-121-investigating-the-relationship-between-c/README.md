# Cosmic Ray Anisotropy Solar-Cycle Modulation

**Project ID**: PROJ-121
**Status**: Active Research

## Overview
This project investigates the relationship between cosmic ray anisotropy and the solar cycle using data from IceCube and the Pierre Auger Observatory, correlated with NOAA solar proxy indices.

## Project Structure
- `code/`: Source code, scripts, and tests
- `data/`: Raw, processed, and result datasets
- `reports/`: Generated LaTeX reports and figures
- `specs/`: Feature specifications and design documents

## Quick Start
1. Install dependencies: `pip install -r code/requirements.txt`
2. Run the full pipeline: `./run_all.sh`
3. Analyze correlation: `python code/analyze_correlation.py`
4. Generate report: `./make_report.sh`

## Prerequisites
- Python 3.11+
- CPU-only environment (no CUDA required)
