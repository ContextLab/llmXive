## Research-question validation

### Phenomenon-vs-method check

**Verdict**: pass

The question asks about a substantive psychological mechanism—whether visual salience modulates attentional drift in moral decision-making—rather than whether a specific algorithm performs well. The aDDM framework serves as a tool to test the phenomenon, not as the phenomenon itself. The core scientific question (does salience bias moral judgments via attention?) is independent of any specific implementation details.

### Circularity check

**Verdict**: pass

The predictor (visual salience) is computed from stimulus properties (images or text heuristics), while the predicted variable (choice outcomes) comes from human decision records in the Moral Machine dataset. These are independent data sources: salience derives from stimulus features, choices derive from human behavior. No circular construction exists.

### Triviality check

**Verdict**: pass

Either outcome would be informative: a positive result would suggest domain-general attentional mechanisms apply to moral decisions (unifying perceptual and moral decision theory), while a null result would suggest moral judgments operate via distinct cognitive mechanisms. Both findings have theoretical value and would be publishable.

### Question-narrowing check

**Verdict**: pass

The question names a domain relationship (visual salience → attentional drift → moral choice outcomes) rather than implementation constraints. It asks about a psychological mechanism, not whether a specific architecture works within budget or hyperparameter limits.

### Overall verdict

**Verdict**: validated

All four checks pass. The research question asks a substantive scientific question about attentional mechanisms in moral decision-making, uses computational modeling as a tool rather than as the question itself, and both positive and null outcomes would be theoretically informative. The project can proceed to initialization.
