- The plan defines the target as 'Residual Bulk Modulus (B_observed - B_Miedema)' and claims this isolates the 'alloying effect'. However, Miedema's model (and similar rule-of-mixtures baselines) is typically parameterized using the exact same elemental properties (atomic radius, electronegativity, valence) that are used to construct the 'Compositional Descriptors' (Mixing Enthalpy, VEC, etc.). If the predictors (X) and the baseline (B_Miedema) share the same physical inputs, the residual target (Y = B_obs - f(X)) is mathematically constrained. The model may simply learn to correct for the baseline's specific functional form rather than discovering independent 'alloying effects,' creating a circular validation where the 'effect' is defined by the failure of the baseline model using the same features. The plan must explicitly verify that the descriptors used for prediction are *orthogonal* to the inputs of the Miedema baseline, or acknowledge that the 'effect' is relative to that specific baseline's physics.
- The plan lists 'Mixing Enthalpy (Miedema's model)' as a descriptor. Miedema's model is a semi-empirical theory that calculates enthalpy based on elemental parameters (electron density, electronegativity). If the model also uses 'atomic radius variance' and 'electronegativity variance' as separate descriptors, there is a high risk of multicollinearity where the 'Mixing Enthalpy' feature is a deterministic function of the other elemental-property-based features. This violates the 'Independence of stated invariants' principle. The plan should clarify if Miedema's enthalpy is used *only* for the baseline subtraction (Target) and *not* as a predictor, or if the predictors are strictly elemental properties that are distinct from the Miedema calculation inputs.
- The plan proposes 'Grouped Bootstrap' sampling by 'alloy system' to prevent leakage. However, in HEA research, 'alloy system' is often defined by the set of constituent elements (e.g., 'CrMnFeCoNi'). If the dataset contains multiple compositions within the same system (e.g., varying stoichiometry), grouping by system is correct. But if the 'system' definition is too broad (e.g., 'transition metals'), leakage occurs. The plan lacks a precise definition of the 'grouping key' (e.g., 'unique set of elements' vs. 'specific stoichiometry'). Without this, the statistical validity of the confidence intervals is undefined. The methodology must specify the exact grouping logic to ensure the bootstrap resamples independent chemical spaces.
