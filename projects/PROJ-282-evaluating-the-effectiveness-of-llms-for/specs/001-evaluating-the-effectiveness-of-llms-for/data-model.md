# Data Model: Evaluating the Effectiveness of LLMs for Identifying Security Vulnerabilities

## Entities

### 1. CodeSnippet
Represents a single unit of source code from the dataset.
- **id**: `string` (Unique identifier, e.g., UUID or dataset index)
- **language**: `string` (Enum: "C", "Python", "JavaScript")
- **source_code**: `string` (The raw code content)
- **ground_truth_label**: `integer` (0: Safe, 1: Vulnerable)
- **ground_truth_category**: `string` (e.g., "SQLi", "BufferOverflow", "XSS")
- **source_dataset**: `string` (e.g., "VulDeePecker")

### 2. FeatureVector
Extracted properties of a `CodeSnippet`.
- **snippet_id**: `string` (Foreign key to CodeSnippet)
- **ast_depth**: `integer` (Max depth of AST)
- **node_count**: `integer` (Total AST nodes)
- **cyclomatic_complexity**: `float` (Complexity score)
- **taint_api_count**: `integer` (Frequency of taint-source APIs)
- **sanitization_present**: `boolean` (True if sanitization found)
- **embedding_similarity_score**: `float` (Cosine similarity to vulnerable patterns)
- **parse_error**: `boolean` (True if AST parsing failed)

### 3. PredictionResult
Output from LLM or Static Analyzer.
- **snippet_id**: `string` (Foreign key)
- **model_name**: `string` (e.g., "TinyLlama-7B", "Bandit")
- **predicted_label**: `integer` (0 or 1)
- **predicted_category**: `string` (Mapped category or "uncertain")
- **is_correct**: `boolean` (True if `predicted_label` == `ground_truth_label`)
- **inference_time_ms**: `float` (Execution time in milliseconds)
- **confidence_score**: `float` (Optional, if model provides probability)
- **truncation_event**: `boolean` (True if input was truncated to fit context window)

### 4. AnalysisMetric
Statistical results.
- **metric_name**: `string` (e.g., "Point_Biserial_r", "Nagelkerke_R2", "McNemar_p")
- **feature_name**: `string` (Feature being analyzed, or "model_comparison")
- **value**: `float` (The calculated statistic)
- **p_value**: `float` (Statistical significance)
- **adjusted_p_value**: `float` (After FDR correction)
- **category**: `string` (Vulnerability category context)

## Relationships
- **CodeSnippet** (1) -- (N) **FeatureVector** (One snippet has one feature vector)
- **CodeSnippet** (1) -- (N) **PredictionResult** (One snippet has predictions from multiple models)
- **FeatureVector** (N) -- (N) **AnalysisMetric** (Metrics aggregate features)

## Data Flow
1. **Ingest**: `CodeSnippet` loaded from Parquet/JSONL.
2. **Transform**: `FeatureVector` generated via `tree-sitter` and embedding model.
3. **Infer**: `PredictionResult` generated by LLM and Static Analyzer.
4. **Aggregate**: `AnalysisMetric` computed from joined `FeatureVector` and `PredictionResult`.
