# Uncovering Correlations Between Processing Conditions and Texture in Rolled Metals

## Overview
This project implements a data-driven pipeline to uncover correlations between processing conditions and texture in rolled metals.

## Structure
- `code/`: Source code for the pipeline
- `data/`: Raw and processed data
- `tests/`: Unit and integration tests
- `docs/`: Documentation and reports

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run the pipeline: `python code/main.py`
