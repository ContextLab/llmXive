# Data Dictionary: Chalcogenide Glass Transition Prediction

This document describes the data schemas, feature definitions, and artifact structures used throughout the pipeline.

## 1. Raw Data Schema
**Source**: Downloaded from the supplementary repository URL specified in `src/data/download.py`.
**File**: `data/raw/chalcogenide_dataset.csv`

| Column Name | Type | Description |
|:--- |:--- |:--- |
| `composition` | string | Chemical formula (e.g., "Ge20Se80", "As30S70"). |
| `Tg` | float | Glass transition temperature in Kelvin. |
| `source` | string | Original publication or database identifier. |

## 2. Processed Features Schema
**Source**: Computed by `src/data/preprocess.py`.
**File**: `data/processed/processed_data.csv`

| Column Name | Type | Description |
|:--- |:--- |:--- |
| `composition` | string | Original chemical formula. |
| `Tg` | float | Target variable. |
| `mean_coordination_number` | float | Average coordination number (CN) of elements in the formula. Calculated using `mendeleev`. |
| `electronegativity_variance` | float | Variance of Pauling electronegativity values weighted by stoichiometry. |
| `atomic_radius_variance` | float | Variance of atomic radii weighted by stoichiometry. |
| `chemical_family` | string | Categorical group (e.g., "Ge-Se", "As-S", "Ge-As-S"). Derived from constituent elements. |
| `sample_size` | int | Number of atoms in the formula unit. |
| `is_valid` | boolean | Flag indicating if all elements were found in the periodic table. |

## 3. Split Artifacts
**Source**: Generated by `src/data/split.py`.
**Directory**: `data/splits/`

- `train_indices.json`: List of integer indices for the training set.
- `test_indices.json`: List of integer indices for the test set.
- `lofo_families.json` (if applicable): List of families used for LOFO CV.

## 4. Model Artifacts
**Source**: Generated by `src/models/train.py`.
**Directory**: `data/models/`

- `linear_baseline.pkl`: Pickled Linear Regression model.
- `gradient_boosting.pkl`: Pickled Gradient Boosting Regressor.
- `lofo_models/`: Directory containing N-1 models for transferability testing.

A Markdown report containing:
- Feature importance ranking (Mean |SHAP|).

## 7. Constants and Properties
**Source**: `src/utils/constants.py` (via `mendeleev`)

- **Coordination Number**: Default valence/coordination based on group number.
- **Electronegativity**: Pauling scale.
- **Atomic Radius**: Calculated atomic radius (pm).
- **Missing Data Handling**: If an element is not found in `mendeleev`, the sample is marked `is_valid=False` and excluded from training.
