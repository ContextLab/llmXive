# Predicting the Impact of Composition on the Glass Transition Temperature of Chalcogenide Glasses

This project implements an automated science pipeline to predict the glass transition temperature (Tg) of chalcogenide glasses based on their chemical composition.

## Prerequisites

- Python 3.11 or higher
- pip

## Setup

1. Install dependencies:
 ```bash
 pip install -r requirements.txt
 ```

2. Run the project initialization script:
 ```bash
 python code/setup_directories.py
 ```

## Project Structure

- `code/`: Source code
 - `src/`: Main source modules
 - `data/`: Data loading, preprocessing, splitting
 - `models/`: Model training and evaluation
 - `utils/`: Utilities, constants, metrics
 - `tests/`: Test suites
- `data/`: Raw and processed data
- `artifacts/`: Model artifacts and reports
- `state/`: Manifest and state tracking
