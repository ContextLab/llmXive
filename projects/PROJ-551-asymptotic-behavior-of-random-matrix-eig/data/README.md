# Data Directory

This directory stores all data artifacts generated by the simulation pipeline.

## Structure
- `raw/`: Raw matrix instances and intermediate states (checksummed)
- `processed/`: Aggregated results, CSV files, and fitted parameters
- `logs/`: Structured JSON logs of simulation runs
- `figures/`: Generated plots and visualizations

## Data Hygiene
All raw data files are checksummed before processing. See `state/checksums.json` for the manifest.
