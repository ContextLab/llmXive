# Quickstart: llmXive Follow-up: OPD Generalization Gap in Unified Diffusion

## Prerequisites
- **Python 3.11** or higher.
- **Git** for **version control**.
- **HuggingFace CLI** (`pip install huggingface_hub`) for dataset/model downloads.
- **Docker** (optional) for isolated environment testing.

## Installation

1. **Clone the repository**:
   ```bash
   git clone <repo-url>
   cd projects/PROJ-913-llmxive-follow-up-extending-qwen-image-2
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r code/requirements.txt
   ```

## Data Acquisition

1. **Download Models**:
   ```bash
   python code/data/download_models.py
   ```
