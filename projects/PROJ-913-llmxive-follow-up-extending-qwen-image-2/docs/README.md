# llmXive: OPD Generalization Gap in Unified Diffusion

This project investigates the generalization gap in Qwen-Image-2.0 models.

## Setup

1. Install dependencies: `pip install -r code/requirements.txt`
2. Run setup: `python code/utils/setup_data_dirs.py`
3. Run pipeline: `python code/data/curate_pilot.py` (and subsequent steps)

## Structure

- `code/`: Source code
- `data/`: Data and outputs
- `tests/`: Tests
- `docs/`: Documentation
