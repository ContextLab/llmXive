## Research Context
This project investigates whether explicit structural motifs explain variance in mutagenicity outcomes compared to global molecular descriptors.

## Validation Standard
Per Marie Curie's review, validation requires:
- Repetition: 5-fold stratified CV repeated 3 times
- Quantification: N > 1000 compounds from verified assays (e.g., PubChem AID 1851) [UNRESOLVED-CLAIM: c_15475a9c — status=not_enough_info]
- Statistical rigor: DeLong's test on Out-of-Fold predictions
