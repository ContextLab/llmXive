# llmXive: Logical Dependency vs. Semantic Curvature in Many-Shot ICL

This project investigates the impact of logical dependency ordering on In-Context Learning (ICL) performance.

## Structure
- `code/src/`: Source code modules
- `code/tests/`: Unit and integration tests
- `data/`: Raw, processed, and results data
- `artifacts/`: Reports and checksums

## Prerequisites
- Python 3.11+
- llama.cpp binary in PATH
- HuggingFace CLI access
