# MemLens CPU Adaptation - Quick Start

This guide runs the simplified reproduction of the MemLens paper's core findings on a CPU-only environment.

## Prerequisites
Ensure you have Python 3.8+ and `matplotlib` installed.
```bash
pip install matplotlib
```

## Run Commands
Execute the simulation script. It will generate CSV data, JSON summaries, and PNG plots in the `data/` and `figures/` directories.

```bash
python code/run_memlens_demo.py
```
