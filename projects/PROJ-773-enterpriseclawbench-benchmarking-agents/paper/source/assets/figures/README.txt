FrontisBench self-contained figure bundle
Each figure folder contains: 
- data.json
- render_<figure>.py
- figure.png
- figure.pdf

Run: python render_<figure>.py
- fig1_pipeline: Figure 1: FrontisBench pipeline overview
- fig2_funnel: Figure 2: Construction funnel
- fig3_benchmark_statistics: Figure 3: Benchmark statistics
- fig4_leaderboard: Figure 4: Main Lite leaderboard
- fig5_role_class_heatmap: Figure 5: Role-class heatmap
- fig6_artifact_type_heatmap: Figure 6: Artifact-type heatmap
- fig7_dimension_profile_heatmap: Figure 7: Dimension profile heatmap
- fig8_skill_transfer_heatmap: Figure 8: Skill-transfer heatmap
- fig9_score_cost: Figure 9: Score vs cost scatter
- fig10_judge_ablation_heatmaps: Figure 10: Judge ablation heatmaps
