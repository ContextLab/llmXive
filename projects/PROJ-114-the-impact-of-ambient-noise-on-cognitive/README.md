# PROJ-114: The Impact of Ambient Noise on Cognitive Flexibility in Remote Workers

## Overview
This project investigates the relationship between ambient noise levels and cognitive flexibility in remote workers using a linear mixed-effects modeling approach.

## Project Structure
```
.
├── code/ # Python source code
├── data/
│ ├── raw/ # Raw data (downloaded/generated)
│ ├── processed/ # Preprocessed data
│ └── results/ # Analysis results
├── docs/ # Documentation
├── specs/ # Feature specifications
├── tests/ # Test suite
├── contracts/ # Data contracts/schemas
├── Snakefile # Snakemake workflow
├── requirements.txt # Python dependencies
└── README.md
```

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run the pipeline: `snakemake --cores 1`
