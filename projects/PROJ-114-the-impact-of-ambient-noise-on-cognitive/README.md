# PROJ-114: The Impact of Ambient Noise on Cognitive Flexibility in Remote Workers

## Project Structure
- `code/`: Source code for data ingestion, modeling, and analysis.
- `data/`: Raw and processed data artifacts.
- `tests/`: Unit and integration tests.
- `contracts/`: JSON Schema definitions.
- `docs/`: Documentation and reports.
- `state/`: Project state tracking.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the pipeline scripts found in `code/scripts/`.
