# Processed Data Directory

This directory contains all intermediate and final processed datasets generated by the analysis pipeline.

## Directory Structure

- `ar_freq_{band}.nc`: Monthly Atmospheric River frequency counts per latitudinal band.
- `corr_fdr_{band}_{season}.nc`: Pearson correlation coefficients, raw p-values, and Benjamini-Hochberg corrected adjusted p-values between AR frequency and Z500 anomalies.
- `sensitivity_summary.csv`: Summary of threshold sensitivity analysis results.

## Regional Domain

All data in this directory is restricted to the regional domain:
- **Latitude**: 20°N to 60°N
- **Longitude**: 100°E to 60°W

## File Naming Conventions

- `{band}`: Latitudinal band identifier (e.g., `20-40N`, `40-60N`, or specific band IDs defined in config).
- `{season}`: Season identifier (e.g., `DJF`, `MAM`, `JJA`, `SON`).

## Data Formats

- **NetCDF (.nc)**: Standard climate data format containing xarray-compatible variables.
- **CSV (.csv)**: Tabular data for sensitivity analysis summaries.

## Generation

These files are generated by running:
```bash
python code/src/cli/run_analysis.py --phase 3
```
