# Figures Directory

This directory contains all visualization outputs generated by the analysis pipeline.

## Directory Structure

- `corr_map_{band}_{season}.png`: Spatial maps of significant Pearson correlation coefficients between AR frequency and Z500 anomalies.

## Regional Domain

All maps are restricted to the regional domain:
- **Latitude**: 20°N to 60°N
- **Longitude**: 100°E to 60°W

## File Naming Conventions

- `{band}`: Latitudinal band identifier (e.g., `20-40N`, `40-60N`).
- `{season}`: Season identifier (e.g., `DJF`, `MAM`, `JJA`, `SON`).
- Files include a content-hash suffix for versioning where applicable.

## Visualization Details

- **Significance Masking**: Non-significant regions (adjusted p-value > 0.05) are masked (transparent/NaN).
- **Color Bar**: Ranges from negative to positive correlation extremes.
- **Projection**: Uses Cartopy for geographic projection.

## Generation

These figures are generated by running:
```bash
python code/src/cli/run_analysis.py --phase 4
```
