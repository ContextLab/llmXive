# PROJ-362: Evaluating the Statistical Validity of Common Ranking Metrics

This project implements permutation tests and power analysis to evaluate the statistical validity of ranking metrics (NDCG@k, MAP) on TREC datasets.

## Structure

- `code/`: Source code modules
- `data/`: Raw and processed data
- `results/`: Output artifacts (null distributions, p-values, plots)
- `specs/`: Design documents
