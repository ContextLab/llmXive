# Data Model: Figure ↔ Data Binding

This document defines the binding between the paper's required figures and the source data files generated by the evaluation pipeline.

## Figure 1: Performance vs. Context Length Scaling Curve

* **Purpose**: Visualize the relationship between model score and log(context_length) to demonstrate the *measured* trend.
* **Data Source**: `results/scaling_report.json` (aggregated from `results/sample_results.json`).
* **Input Schema**:
 ```json
 {
 "context_lengths": [32000, 64000, 128000, 256000, 512000],
 "mean_scores": [0.85, 0.82, 0.78, 0.65, 0.55],
 "std_scores": [0.05, 0.06, 0.07, 0.10, 0.12],
 "slope_coefficient": -0.00002,
 "r_squared": 0.85,
 "trend_classification": "sublinear"
 }
 ```
* **Generation Script**: `src/eval/scaling_analysis.py`
 * Reads: `results/sample_results.json`
 * Writes: `results/scaling_report.json`
 * Logic: Fits OLS regression of `score` ~ `log(context_length)`.
 * **Variance Handling**: If `std_scores` > 0.15 for any point, `trend_classification` is forced to "highly variable".

## Figure 2: Generalization Retention Rate Bar Chart

* **Purpose**: Compare *measured* performance at 128K (baseline) against 256K and 512K.
* **Data Source**: `results/validation_report.md` (derived from `results/sample_results.json`).
* **Input Schema**:
 ```json
 {
 "baseline_context": 128000,
 "baseline_score": 0.78,
 "retention_rates": {
 "256k": 0.83,
 "512k": 0.71
 },
 "threshold_met": false
 }
 ```
* **Generation Script**: `src/eval/validate_results.py`
 * Reads: `results/sample_results.json`
 * Writes: `results/validation_report.md` (and intermediate JSON for plotting)
 * Logic: Calculates `score_at_XK / score_at_128K`.

## Figure 3: Reproduction Delta Plot

* **Purpose**: Show the *measured* difference between reproduced scores and the original paper's claimed scores.
* **Data Source**: `results/sample_results.json` vs. hardcoded baseline values from the original paper abstract.
* **Input Schema**:
 ```json
 {
 "task_type": "long-document VQA",
 "reproduced_score": 0.78,
 "original_claim": 0.80,
 "deviation_percent": 2.5
 }
 ```
* **Generation Script**: `src/eval/report_generator.py` (or inline in `run_cpu_eval.py` for final report).
 * Reads: `results/sample_results.json`
 * Writes: `results/final_report.md` (with embedded plot data)
 * Logic: `(reproduced - original) / original * 100`.

## Data Provenance & Manifest

All source data files must be referenced in `data/manifest.json`:

* **Dataset**: `yubo2333/MMLongBench-Doc`
* **Split**: `test`
* **Checksum**: SHA-256 of the downloaded dataset files.
* **Version**: Commit hash or tag of the dataset at the time of download.
