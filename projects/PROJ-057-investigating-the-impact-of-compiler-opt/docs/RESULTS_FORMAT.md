# Output Data Format Specification

This document describes the format of the output artifacts generated by the benchmark suite.

## 1. Raw Logs (`data/intermediates/raw_logs/{config_id}.jsonl`)

Each line is a JSON object representing a single run configuration.

```json
{
 "config_id": "matmul_512x512_O2",
 "kernel_type": "matmul",
 "flags": ["-O2"],
 "dimensions": [512, 512],
 "iterations": 1000,
 "median_latency_us": 1234.56,
 "p95_latency_us": 1300.12,
 "downsampled": false,
 "status": "SUCCESS"
}
```

## 2. Stability Metrics (`data/results/stability_metrics.csv`)

CSV file containing stability analysis results.

| Column | Type | Description |
|:--- |:--- |:--- |
| `config_id` | str | Unique identifier for the configuration |
| `kernel_type` | str | Type of kernel (matmul, softmax, layernorm) |
| `l2_error` | float | L2 relative error compared to reference |
| `max_diff` | float | Maximum absolute difference |
| `status` | str | "STABLE" or "UNSTABLE" (if error > 1e-5) |

## 3. Aggregated Results (`data/results/aggregated.csv`)

Final aggregated data used for statistical analysis and plotting.

| Column | Type | Description |
|:--- |:--- |:--- |
| `config_id` | str | Configuration ID |
| `median_latency` | float | Median latency in microseconds |
| `max_abs_diff` | float | Max absolute difference |
| `is_stable` | bool | True if error <= 1e-5 |
| `is_downsampled` | bool | True if run was downsampled due to memory |

## 4. Manifest (`data/manifest.json`)

JSON file containing SHA-256 hashes of all generated artifacts for reproducibility.

```json
{
 "artifacts": [
 {
 "path": "data/results/aggregated.csv",
 "sha256": "abc123..."
 },
 {
 "path": "data/results/pareto_frontier_final.png",
 "sha256": "def456..."
 }
 ],
 "timestamp": "2023-10-27T10:00:00Z"
}
```

## 5. Visualizations

- **`pareto_frontier_exploration.png`**: Scatter plot of `median_latency` (x) vs `max_abs_diff` (y). Includes downsampled runs (distinct marker) and excludes unstable runs.
- **`pareto_frontier_final.png`**: Strict Pareto frontier excluding all unstable runs (error > 1e-5).
