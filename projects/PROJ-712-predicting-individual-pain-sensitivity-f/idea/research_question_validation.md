## Research-question validation

### Phenomenon-vs-method check

**Verdict**: pass

The question asks whether characteristics of the brain’s intrinsic microstate dynamics (a neurophysiological phenomenon) are predictive of a behavioral pain‑sensitivity measure (heat‑pain threshold). It does not hinge on any particular algorithmic implementation or resource constraint, so it is a substantive scientific question about the brain‑behavior relationship.

### Circularity check

**Verdict**: pass

Predictor data come from resting‑state EEG recordings processed into microstate metrics; the predicted variable is an independently obtained psychophysical heat‑pain threshold. These sources are distinct (neurophysiological vs behavioral), so the relationship is not mechanically guaranteed.

### Triviality check

**Verdict**: pass

There is no strong prior consensus that microstate dynamics encode individual pain thresholds. Demonstrating a significant predictive link would provide a novel biomarker; a null finding would indicate that microstates alone are insufficient, both outcomes advancing knowledge.

### Question-narrowing check

**Verdict**: pass

The question frames a domain‑level inquiry—how a specific aspect of brain activity relates to pain sensitivity—rather than imposing a constraint on a particular method’s performance.

### Overall verdict

**Verdict**: validated
