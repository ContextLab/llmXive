# Code Directory

This directory contains the main Python modules for the pipeline.

- `config.py`: Configuration management.
- `data_models.py`: Base data models.
- `ingestion.py`: Data download and filtering.
- `features.py`: Linguistic feature extraction.
- `inference.py`: LLM inference.
- `labeling.py`: Adherence labeling.
- `modeling.py`: Statistical analysis.
- `validation.py`: Validation logic and logging.
- `annotation.py`: Human annotation pipeline.
