# Data Directory

This directory stores all data artifacts.

## Subdirectories

- `raw/`: Original downloaded datasets (e.g., MedMisBench).
- `interim/`: Intermediate processed data (e.g., annotated responses).
- `processed/`: Final datasets ready for analysis (e.g., feature-rich CSVs).
- `results/`: Output from statistical modeling and analysis.

**Note**: Large data files should be gitignored. Only small sample files or metadata should be committed.
