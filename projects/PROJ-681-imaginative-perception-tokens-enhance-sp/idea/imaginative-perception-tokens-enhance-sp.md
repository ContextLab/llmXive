# Imaginative Perception Tokens Enhance Spatial Reasoning in Multimodal Language Models

**Field**: computer science

## Research question

What properties of representation mechanisms (implicit inference, explicit token generation, or external memory retrieval) enable robust spatial inference in vision-language models when critical information is missing from the input, and how do these mechanisms generalize across different spatial reasoning benchmarks?

## Motivation

Vision-language models (VLMs) frequently fail when spatial reasoning requires inferring information absent from the input image. Recent work introduces "imaginative tokens" to address this, but comparative effectiveness across representation strategies remains unquantified. This project isolates representation mechanisms empirically to determine which enables robust spatial inference under missing-information constraints, informing architectural choices for downstream VLM applications.

## Related work

- [Imaginative Perception Tokens Enhance Spatial Reasoning in Multimodal Language Models](https://arxiv.org/abs/2606.03988) — Establishes the foundational concept of imaginative tokens for VLMs facing unobservable spatial information, serving as the primary baseline for this ablation study.
- [Perception, Reason, Think, and Plan: A Survey on Large Multimodal Reasoning Models](https://arxiv.org/abs/2505.04921) — Identifies spatial inference as a key capability gap in current multimodal reasoning systems, motivating the focus on missing-information scenarios.
- [Imagine in Space: Exploring the Frontier of Spatial Intelligence and Reasoning Efficiency in Vision Language Models](https://arxiv.org/abs/2511.13782) — Documents current VLM spatial reasoning limitations and provides efficiency benchmarks for measuring improvement across different architectures.
- [Visual Thoughts: A Unified Perspective of Understanding Multimodal Chain-of-Thought](https://arxiv.org/abs/2505.15510) — Shows how multimodal reasoning chains enhance interpretability on complex visual tasks, offering methodological parallels for token-based reasoning strategies.

## Expected results

We expect explicit token generation strategies to outperform implicit inference methods on tasks requiring multi-step spatial deduction, with measurable accuracy differences across benchmarks. Improvements should be statistically significant (p < 0.05) with at least 3 random seed replicates, and effects should generalize across at least two distinct spatial reasoning benchmarks (e.g., AI2-THOR and EgoDir).

## Methodology sketch

- Download spatial reasoning benchmarks: AI2-THOR ProcTHOR subset (https://ai2thor.allenai.org), EgoDir (https://github.com/rohan-singh/egodir), and Multiview Counting; limit to N=200 validation samples per benchmark to fit 6h CPU job.
- Load pretrained VLM baselines (LLaVA-1.5-7B) via HuggingFace Transformers with 4-bit quantization (bitsandbytes) and CPU offloading to stay within 7GB RAM.
- Implement three representation strategies: (1) implicit inference (standard VLM), (2) explicit token generation (imaginative tokens), (3) external memory retrieval (vector store of spatial facts).
- For each benchmark and strategy, run inference with 3 random seeds; record accuracy per task and inference latency.
- Compute accuracy differences between strategies and baseline; perform paired t-tests with Bonferroni correction for multiple comparisons.
- Generate bootstrap confidence intervals (1000 resamples) for effect sizes to assess generalization across benchmarks.
- Validate token representations against independent benchmark ground truth (task labels from test set, not derived from token embeddings or training data) to ensure evaluation independence.
- Document all hyperparameters, seeds, and evaluation splits in reproducible YAML configuration files.
- Use CPU-only inference with batch size ≤4 to maintain memory constraints and prevent OOM errors on free-tier runners.
