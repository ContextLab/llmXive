# Schema Documentation

The following JSON/YAML schemas are provided in this directory:

- `knot_record.schema.yaml` – defines the `KnotRecord` data model.
- `invariants_and_regression.schema.yaml` – defines the `InvariantsDataset` and `RegressionModel` structures.
- `additional_schemas.yaml` – auxiliary schema definitions.

These schemas are used throughout the project for data validation and contract enforcement.
