# Narrative on Combined Invariant Intuition

Imagine holding a piece of string, tying it into a knot, and then trying to describe that knot to someone who cannot see it.  One can count the crossings, trace the braid, but ultimately this is a story about entanglement.

The classical invariants—crossing number and braid index—each capture a distinct facet of a knot’s geometry.  The crossing number measures combinatorial complexity, while the braid index reflects the minimal number of strands needed to represent the knot as a closed braid.

Together they suggest a richer, two‑dimensional view of knot complexity, analogous to how a Fourier series combines sine and cosine components to describe a signal.  In this project we formalize this intuition by constructing a composite metric that jointly leverages both invariants.
