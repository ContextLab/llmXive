# Provenance and Licensing

All source code in this repository is released under the **MIT License** (see the top‑level `LICENSE` file).

Data provenance is recorded in `data/provenance.yaml` and summarized in `PROVENANCE.md`.  These documents detail the original sources of each dataset, any preprocessing steps performed, and the licensing terms associated with the source data.

For a comprehensive overview of how licensing and provenance are managed throughout the project, see the existing documentation in `docs/reproducibility/license_and_provenance_overview.md`.
