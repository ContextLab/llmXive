# Reproducing the Pipeline

This project requires **Python 3.10** (or later). Verify your Python version with:

```bash
python --version
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the quick‑start script to execute the full analysis pipeline:

```bash
python -m code.reproducibility.quickstart_validator
```

For more detailed usage, refer to the existing `docs/reproducibility/quickstart.md`.
