# Reproducibility Artifacts Overview

This directory contains all artifacts required to reproduce the results
presented in the project **Quantifying the Complexity of Knot Diagrams**.

## Key items

- **`checksums/`** – Files documenting cryptographic hashes for all data
  assets (`checksums.csv`, `checksums.json`, `checksums.sha256`).
- **`logs/`** – Detailed execution logs (`reproducibility.log`, `logs.jsonl`).
- **`final_report.md`** – Consolidated reproducibility report summarizing
  validation status, data quality, and methodology.
- **`quickstart.md`** – Step‑by‑step guide to run the full pipeline.

Refer to the individual markdown files in this folder for deeper explanations.
