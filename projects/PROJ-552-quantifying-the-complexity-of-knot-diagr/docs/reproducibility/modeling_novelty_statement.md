# Modeling Novelty Statement

While the regression analyses (linear, polynomial, logarithmic) and residual‑family identification are standard tools, our work integrates these models into a predictive framework that estimates knot diagram complexity for *unseen* knot families. By employing cross‑validation, model selection criteria (AIC/BIC), and extrapolation techniques, we move beyond descriptive statistics to enable **prediction** of complexity metrics for newly generated knot diagrams. This predictive capability supports hypothesis generation and guides future data collection efforts.
