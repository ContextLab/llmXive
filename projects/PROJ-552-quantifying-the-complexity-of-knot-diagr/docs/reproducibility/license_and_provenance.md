## Provenance

See the consolidated provenance information in [PROVENANCE.md](../PROVENANCE.md).

Each dataset includes a provenance YAML file (`data/provenance.yaml` and `data/raw/*.provenance.yaml`) that records the source, processing steps, timestamps, and responsible contributors.  These files ensure reproducibility and traceability of all results presented in the project.
