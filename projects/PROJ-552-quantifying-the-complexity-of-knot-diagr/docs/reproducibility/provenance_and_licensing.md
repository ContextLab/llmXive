# Data Provenance and Licensing

All knot data used in this project are derived from the **Knot Atlas**
repository (https://katlas.math.toronto.edu) which is released under the
Creative Commons Attribution 4.0 International (CC BY 4.0) license.  The
original dataset is cited as:

> **Knot Atlas** – Bar-Natan, D., et al. (2014). *The Knot Atlas*.  
> https://katlas.math.toronto.edu

Our processed CSV files (`data/processed/*.csv`) are transformations of the
original JSON data.  The transformations preserve the original licensing terms;
therefore the derived data are also made available under CC BY 4.0.  Users must
provide appropriate attribution when redistributing or publishing results that
use these data.

The source code in the `code/` directory is released under the MIT License, as
stated in the top‑level `LICENSE` file.  When re‑using code, please retain the
license notice and include a citation to this repository.

## How to Cite

If you use the data or software from this project in your research, please cite
the repository and the original Knot Atlas source.  A suggested BibTeX entry is:

```bibtex
@misc{knot_complexity_2026,
  author = {Your Name and Collaborators},
  title  = {Quantifying the Complexity of Knot Diagrams},
  year   = {2026},
  url    = {https://github.com/yourrepo/PROJ-552-quantifying-the-complexity-of-knot-diagr},
  note   = {Version X.Y.Z},
}
```

Please also include the Knot Atlas citation shown above.
