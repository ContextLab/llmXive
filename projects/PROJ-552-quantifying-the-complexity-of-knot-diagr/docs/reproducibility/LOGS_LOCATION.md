# Log Files Location

All operational log files are stored in this directory (`docs/reproducibility/`) as required by **FR‑007**.

The following logs are now located here:

- `reproducibility.log`
- `logs.jsonl`
- `logs.json`
- `operation_logs.jsonl`

These files have been moved from the previous top‑level `logs/` directory to satisfy the reproducibility documentation requirement.
