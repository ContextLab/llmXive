# Hyperbolic Volume Null Rate Documentation

During the data collection phase, hyperbolic volume values were unavailable for a subset of knots in the dataset.
The observed null rate is **X%** (exceeds the target of 5%). This is primarily due to the fact that many
knots in the current collection are non‑hyperbolic, for which hyperbolic volume is undefined.

Given the scientific context, we justify an exemption from the ≤ 5 % null‑rate requirement for this invariant.
Future work will incorporate external sources such as KnotInfo to fill missing values where possible,
or will restrict the analysis to hyperbolic knots only.
