# Provenance and Licensing Overview

This project follows clear provenance and licensing policies to ensure that all
research artifacts are reusable and properly attributed.

* **Source Code** – All code in the repository is released under the MIT License
  (see the top‑level `LICENSE` file).
* **Data** – The primary knot dataset originates from the Knot Atlas and is
  provided under the Creative Commons Attribution 4.0 International (CC‑BY‑4.0)
  license.  Detailed licensing information is available in
  `docs/reproducibility/knot_atlas_data_license.md`.
* **Provenance Tracking** – The `PROVENANCE.md` file documents the origin,
  transformation steps, and versioning of each dataset used in the analyses.
* **Reproducibility Artifacts** – Checksums, logs, and derivation notes are
  generated automatically (see `docs/reproducibility/checksums.md` and
  `docs/reproducibility/derivation_notes.md`).

For a complete description of licensing terms and provenance records, please
refer to the linked documents.
