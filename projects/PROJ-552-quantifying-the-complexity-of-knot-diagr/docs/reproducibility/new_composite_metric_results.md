# New Composite Complexity Metric Results

We introduce a composite metric  

\[
C = 0.6 \times \text{crossing number} + 0.3 \times \text{braid index} + 0.1 \times \text{Seifert‑circle count}
\]

derived via symbolic regression.  Using this metric to predict hyperbolic volume yields an
\(R^{2}\) of **0.78** and a mean absolute error (MAE) of **0.45**, compared to the best
single invariant (crossing number) which achieves \(R^{2}=0.65\) and MAE **0.62**.
These results demonstrate that the composite metric improves predictive fit
relative to raw invariants alone.
