# Entry Points

The main entry points for this project are:

- `python -m code.main` – runs the primary analysis pipeline.
- `code/main.py` – executable script for the full workflow.
- Quick‑start command: `proj-quantify-knot-complexity` (installed via the package's console script) – runs a default analysis on the Knot Atlas dataset.

These commands provide convenient ways to start the analysis without needing to navigate the source tree directly.

For more detailed usage information, see the **Quickstart** documentation.
