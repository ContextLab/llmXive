# Main Entry Points

The project provides the following primary ways to run the analysis:

- **Python module entry point**: `python -m code` (the `__main__` module) which invokes the main pipeline.
- **Script entry point**: `code/main.py` – the main executable script.
- **Quick‑start command**: `proj-quantify-complexity` (installed via the package's console script) – runs the default analysis with recommended settings.

See the quick‑start guide for usage examples.
