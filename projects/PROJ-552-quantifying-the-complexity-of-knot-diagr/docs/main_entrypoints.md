# Main Entry Points

This project provides two primary ways to run the analysis:

1. **Direct script execution**: `python -m code.main` (or `code/main.py` if executed as a script).
2. **Quick‑start command**: `knot-complexity-quickstart` (installed via the package entry point).

Both commands launch the full analysis pipeline with default settings. See the
quick‑start guide for additional options.
