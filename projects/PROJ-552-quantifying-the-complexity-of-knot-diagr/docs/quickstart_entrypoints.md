# Quick‑start Entry Points

The project provides the following primary ways to run the analysis:

- **Direct script**: `python -m code.main` – executes the main pipeline defined in `code/main.py`.
- **Quick‑start command**: `proj-quantify-complexity` – after installing the package, this console script runs the default analysis workflow.

Both commands will process the knot dataset and generate the standard reproducibility reports.
