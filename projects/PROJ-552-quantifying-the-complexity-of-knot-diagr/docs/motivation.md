# Project Motivation

Imagine holding a piece of string, tying it into a knot, and then trying to describe that knot to someone who cannot see it.  One can count crossings or trace the braid, but fundamentally this is telling a story about entanglement.

The crossing number and braid index are classical invariants, each capturing a different facet of a knot’s geometry.  Together they may form a richer descriptor, much like how a single Fourier component contributes to a full signal. This project formalizes that intuition by exploring combined invariants and their predictive power.
