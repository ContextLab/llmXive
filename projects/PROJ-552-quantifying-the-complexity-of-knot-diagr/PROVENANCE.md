# Provenance
All code and data in this repository are provided under the MIT License (see LICENSE).
Third‑party components are covered by the licenses listed in LICENSES.md.
The provenance of each dataset is documented in `data/provenance.yaml`.

This project tracks provenance of data, code, and analysis artifacts.

## Data provenance
- Raw knot atlas data: `data/raw/knot_atlas_raw.json` with provenance recorded in `data/raw/knot_atlas_raw.provenance.yaml`.
- Processed datasets are derived from the raw data and their provenance is summarized in `data/provenance.yaml`.

## Code provenance
- All source code under `code/` is version‑controlled via Git. The repository history records changes to analysis scripts, reproducibility tools, and utilities.

## Reproducibility documentation
- Detailed provenance notes are consolidated here to avoid duplication across multiple `docs/reproducibility/*.md` files.

For more details, see the individual provenance files referenced above.
