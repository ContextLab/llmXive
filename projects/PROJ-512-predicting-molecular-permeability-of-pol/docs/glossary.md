# Glossary

- **PolymerGraph**: A data structure representing a polymer repeat unit as a graph with nodes (atoms) and edges (bonds).
- **SMILES**: Simplified Molecular Input Line Entry System, a string representation of a molecule.
- **Murcko Scaffold**: The core framework of a molecule, excluding side chains. Used for splitting data.
- **GNN**: Graph Neural Network, a model designed to operate on graph-structured data.
- **ECFP4**: Extended Connectivity Fingerprints, a type of molecular fingerprint used for baselines.
- **VIF**: Variance Inflation Factor, a measure of multicollinearity in regression models.
- **Barrer**: Unit of gas permeability.
- **Log-Permeability**: The base-10 logarithm of the permeability coefficient, used as the target variable.
- **FR-001**: Functional Requirement 001, which restricts the model to 2D features only.
- **Topology Control**: A baseline method where edge connections are randomized to test if the model learns structure or just composition.
