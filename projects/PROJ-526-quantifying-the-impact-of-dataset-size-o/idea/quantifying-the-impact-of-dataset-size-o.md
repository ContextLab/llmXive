# Quantifying the Impact of Dataset Size on the Accuracy of Machine Learning Models for Predicting Material Properties

**Field**: physics

Can the accuracy of machine learning models predicting material properties (e.g., band gap, bulk modulus) be reliably quantified as a function of the size of the training dataset? This project proposes to re-analyze publicly available materials property datasets (e.g., Materials Project, AFLOW) using standard ML regression algorithms (e.g., Random Forests, Gradient Boosting) and systematically varying the training set size. The goal is to establish scaling laws relating dataset size to prediction error, providing guidance for future materials discovery efforts.  This analysis will focus on datasets readily downloadable via HTTPS and computationally feasible to train models on within the GHA runner constraints.
