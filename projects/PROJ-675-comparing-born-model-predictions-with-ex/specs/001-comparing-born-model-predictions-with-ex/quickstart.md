# Quickstart Guide: Born Model Solvation Energy Comparison

This guide provides minimal run instructions for the core scripts and test suite of the PROJ-675 project.

## Prerequisites

- Python 3.11 or later
- pip package manager
- Unix-like shell (bash, zsh, or WSL on Windows)

## Environment Setup

1. Create and activate a virtual environment:

 ```bash
 python -m venv venv
 source venv/bin/activate # On Windows: venv\Scripts\activate
 ```

2. Install dependencies from requirements.txt:

 ```bash
 pip install -r requirements.txt
 ```

## Running the Data Compiler

The data compiler fetches experimental solvation energies, ionic radii, and solvent dielectric constants from public databases.

```bash
python code/data_compiler.py
```

**Output**: `data/experimental_solvation.csv`, `data/parameters.csv`, `data/metadata.json`

**Notes**:
- Requires network access to fetch data from NIST and other sources
- May take 2-5 minutes depending on network conditions
- Retry logic is built-in for transient failures

## Running the Born Calculator

The Born calculator computes theoretical solvation free energies using the Born equation.

```bash
python code/born_calculator.py
```

**Output**: `data/born_predictions.csv`

**Notes**:
- Reads input data from `data/experimental_solvation.csv` (generated by data_compiler.py)
- Supports crystal and hydrated ionic radius types via command-line argument
- Includes unit conversion utilities (Å to m, kcal/mol to J)

**Command-line options**:
```bash
python code/born_calculator.py --radius-type crystal # or hydrated
python code/born_calculator.py --temperature 298.15 # K, default 298.15
```

## Running Tests

Run the complete test suite with pytest:

```bash
pytest tests/ -v
```

Run specific test categories:

```bash
# Contract tests (schema validation)
pytest tests/contract/ -v

# Unit tests
pytest tests/unit/ -v

# With coverage report
pytest tests/ --cov=code --cov-report=term-missing
```

## Validation Checks

After running the data compiler and calculator, verify outputs:

```bash
# Check data validation (requires ≥30 ion-solvent pairs)
python code/validate_data.py

# Check Born calculator validation (requires ≤1% relative error on reference set)
python code/validate_born.py
```

## Troubleshooting

### Import Errors
If you encounter `ModuleNotFoundError`, ensure you are running from the project root:

```bash
cd /path/to/project/root
python code/data_compiler.py
```

### Network Failures
If data fetching fails, check your network connection and firewall settings. The data compiler includes retry logic (3 attempts with exponential backoff).

### Schema Validation Failures
If contract tests fail, verify that the generated CSV files match the schema defined in `contracts/`.
