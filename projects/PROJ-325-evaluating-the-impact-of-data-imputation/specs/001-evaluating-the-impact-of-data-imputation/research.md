## Statistical Methodology & Rigor

### Variance Estimation
- **Method**: Taylor Series Linearization (default) and Jackknife Repeated Replication (JRR) for robustness. Implemented via `statsmodels.survey` utilities (Task T009).  
- **Design‑Integrity**: If any of `weight`, `psu`, or `strata` are missing, the analysis for that variable is **aborted** (Principle VI).  

### Imputation Strategies
1. **Complete‑Case (CC)** – Excludes rows with missing `target`.  
2. **Single Mean Imputation** – Replaces missing values with column mean; computes pooled variance.  
3. **MICE** – Implemented per **Task T021** using `miceforest`:  
   - Multiple independent chains, each with its own seed (`base_seed + chain_id`).  
   - `max_iter=1000` to ensure convergence.  
   - First **500** imputations per chain are discarded as burn‑in; the remaining `m` imputations are pooled via Rubin’s Rules.  
   - For binary outcomes, Predictive Mean Matching with a `RandomForestRegressor` is used, and convergence is assessed via R‑hat < 1.05 (Task T026).  

### Bias & Success Criteria
- **Synthetic Bias**: `(estimated_variance - true_variance) / true_variance`.  
- **Real‑Data Metric**: *Relative efficiency* defined as `abs(estimated_variance - jackknife_variance) / jackknife_variance`. This measures how close an imputation‑based variance estimate is to a robust design‑based Jackknife estimate without claiming absolute bias.  
- **SC‑002 Check**: Compute `ratio_to_single = deviation_MICE / deviation_Single`. The criterion passes if `ratio_to_single ≤ 0.8`. Implemented in **Task T024**.  
- **Multiplicity**: Holm‑Bonferroni correction applied to all paired t‑tests (Task T013). The SC‑002 ratio is a point‑estimate comparison and does **not** require multiplicity adjustment.  

### Edge‑Case Handling
- Missingness > 30 % → variable skipped, warning logged.  
- PSU = 1 clusters → warning + “potentially unstable” flag (Task T009b).  
- MICE convergence failure → retry with new seed up to 3 attempts (Task T027); if still failing, record `status: warning`.
