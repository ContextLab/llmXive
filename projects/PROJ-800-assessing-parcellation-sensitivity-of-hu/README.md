# PROJ-800: Assessing Parcellation Sensitivity of Hub Resilience in Healthy Connectomes

## Overview
This project assesses how the definition of "hubs" in healthy connectomes changes
when using different brain parcellations (AAL-90, Schaefer-200, Schaefer-400).

## Project Structure
The project follows a standard scientific computing layout:
- `data/`: Raw, processed, and result data
 - `raw/`: Downloaded fMRI NIfTI files
 - `processed/`: Parcellated time series and adjacency matrices
 - `results/`: Centrality scores, overlap metrics, and reports
- `code/`: Python modules for the pipeline
- `tests/`: Unit and integration tests
- `specs/`: Design documents and requirements

## Setup
1. Ensure Python 3.11+ is installed.
2. Install dependencies: `pip install -r requirements.txt`
3. Initialize the project directory structure: `python code/setup_directories.py`
