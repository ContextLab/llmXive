# Assessing Parcellation Sensitivity of Hub Resilience in Healthy Connectomes

**Field**: neuroscience

This project evaluates the robustness of graph-theoretical hub identification across multiple standard brain parcellation schemes using publicly available processed connectivity matrices. While network neuroscience relies heavily on predefined atlases, the stability of key metrics like degree centrality and betweenness across resolutions remains under-characterized in healthy populations. We will download precomputed correlation matrices from open repositories, apply three distinct parcellation resolutions, and compute hub overlap statistics. This analysis matters because inconsistent hub identification undermines cross-study comparisons and biomarker development. The approach avoids raw signal preprocessing, relying solely on matrix algebra suitable for CPU execution within one hour. Results will quantify methodological variance, informing best practices for future connectomic studies and ensuring that reported network abnormalities reflect biology rather than atlas choice.
