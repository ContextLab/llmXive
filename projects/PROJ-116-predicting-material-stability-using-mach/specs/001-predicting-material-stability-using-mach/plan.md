## Summary

This project implements a machine learning pipeline to predict material stability (formation energy) and classify metastable phases. The approach establishes a baseline using bulk compositional descriptors (Magpie features) and evaluates the incremental value of local coordination features (Voronoi statistics, bond-length histograms) derived from crystal structures. The pipeline targets Li-rich rock-salt oxides from the OQMD dataset, utilizing Gradient Boosting Regressors for prediction and `pymatgen` for convex hull distance calculations. The entire workflow is constrained to a CPU-only environment (2 cores, 7 GB RAM) to ensure compatibility with free-tier CI runners.

## Complexity Tracking

| Violation | Why Needed | Simpler Alternative Rejected Because |
| :--- | :--- | :--- |
| **Separate Training Scripts** | Allows isolation of baseline vs. augmented models for clear delta calculation. | Single monolithic script would obscure the specific contribution of local features. |
| **Sensitivity Analysis Sweep** | Required by FR-005 to validate the 0.05 eV/atom threshold robustness. | Fixed threshold would fail to address DFT uncertainty and community standards. |
| **Local Feature Engineering** | Necessary to test the hypothesis that local disorder captures instability. | Bulk-only features (Magpie) are insufficient for the specific research question on metastability. |
| **Permutation Test** | Required to handle non-normal error distributions in materials science. | Parametric t-tests may yield invalid results due to heavy-tailed error distributions. |
| **VIF Analysis** | Required to quantify collinearity between bulk and local features. | Without VIF, claims of "independent effects" may be statistically invalid. |
