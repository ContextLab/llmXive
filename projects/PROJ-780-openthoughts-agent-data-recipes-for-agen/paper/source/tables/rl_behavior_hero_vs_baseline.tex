\caption{\textbf{Two RL runs on the same pipeline learn opposite behavioral policies.}

On \emph{every} behavioral axis the two runs move in opposite directions: the hero policy
\emph{expands} (more turns, tokens, tool calls, thinking, self-correction) while the
