## Unresolved concerns

- The plan proposes a 'Zero-Inflated Gamma (ZIG) or Hurdle meta-analysis' but fails to define the meta-analytic unit. ZIG/Hurdle models yield two distinct coefficients (log-odds of zero-inflation and log-scale of positive amounts). The plan does not specify whether the meta-analysis will pool the exclusion effect on the *probability* of donating, the *amount* donated (conditional on donating), or both. Pooling only one component while ignoring the other risks a 'construct validity' failure where the total causal effect is misestimated.
- The sensitivity analysis plan specifies sweeping a 'zero-inflation threshold $	heta \in \{0.01, 0.05, 0.10\}$'. This is a methodological error. In ZIG/Hurdle models, the zero-inflation is a *modeled parameter* (probability), not a data threshold. Sweeping a threshold implies manually reclassifying data points (e.g., 'if amount < 0.05, treat as zero'), which contradicts the use of a generative model designed to handle structural zeros. This confounds the model's internal estimation with arbitrary data binning, invalidating the robustness check.
- The plan states: 'A post-hoc power analysis is not feasible for meta-analysis without individual participant data (IPD) power curves'. This is incorrect. Standard meta-analysis power calculations (e.g., using the `metafor` package in R or equivalent in Python) can estimate the power to detect a pooled effect size given the number of studies, their sample sizes, and observed heterogeneity ($I^2$). Relying on a 'sample size filter' (n<5) without a formal power assessment for the *meta-analytic* effect risks a Type II error (false negative) if the aggregated data is still underpowered to detect the expected small-to-medium psychological effect.

- The plan proposes a 'Sensitivity Sweep' on the 'zero-inflation threshold' ($\theta \in \{0.01, 0.05, 0.10\}$) to verify model stability. This is a methodological category error. In Zero-Inflated Gamma (ZIG) or Hurdle models, the 'zero' state is structural (a distinct latent class), not a continuous threshold applied to the outcome variable. The model estimates the probability of belonging to the zero-generating process ($\pi$) via a logistic link function; it does not use a scalar $\theta$ to classify zeros. Sweeping a non-existent parameter $\theta$ cannot validate the model's assumptions and renders the sensitivity analysis scientifically invalid.
- The plan defines the sensitivity metric as 'Variance of $\beta$ across sweeps must be < 10%'. This constitutes a fundamental flaw in the research design, as the proposed validation step cannot empirically test the robustness of the effect size.
- The plan conflates 'Zero-Inflated' modeling with 'Thresholding'. ZIG models assume two processes: one generating structural zeros and one generating positive values. The sensitivity analysis should instead test the robustness of the *link function* (e.g., logit vs. probit) or the *distributional assumption* of the positive component (e.g., Gamma vs. Log-Normal), not a 'zero-inflation threshold' which implies a continuous cutoff that does not exist in the model's mathematical formulation.
