# PROJ-008: Mindfulness Components and Delivery Formats in ASD Social Skills

A meta-analysis investigating the efficacy of mindfulness-based interventions for improving social skills in children with Autism Spectrum Disorder (ASD).

## Project Structure

- `code/`: Source code modules
 - `data/`: Data collection and cleaning
 - `analysis/`: Statistical analysis
 - `viz/`: Visualization
 - `utils/`: Utilities
- `data/`: Data storage
 - `raw/`: Downloaded raw data
 - `processed/`: Cleaned and processed data
 - `figures/`: Generated plots
- `tests/`: Test suite
- `specs/`: Feature specifications
- `contracts/`: Data schema contracts
- `docs/`: Documentation
