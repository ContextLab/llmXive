# Social Memory Networks: Modeling Collective Remembering in Multi-Agent LLMs

Study how memories are shared, distorted, and consolidated across networks of interacting LLMs, mimicking human social memory processes. This project would explore collective intelligence and memory convergence in AI systems.

Research components:
- Implement memory sharing protocols between LLM agents
- Study how narratives evolve through retelling chains
- Model social contagion of false memories
- Test collaborative recall and transactive memory
- Explore emergence of 'cultural' memories in agent societies

Applies social psychology of memory to multi-agent AI systems.
