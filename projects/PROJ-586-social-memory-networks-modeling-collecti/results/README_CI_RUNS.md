# CI Pipeline Run Results

This directory contains artifacts generated by the automated CI pipeline execution
for the Social Memory Networks project.

## Contents

- `ci_run_report_*.json`: JSON reports containing detailed runtime, memory, and
 disk usage metrics for each pipeline run.
- `ci_run_log_*.txt`: Human-readable logs summarizing the execution flow.
- `results_*.csv`: Experimental output data from the simulation runs.
- `scaling_plot.pdf`: Visualization of agent scaling analysis.

## Metrics Recorded

Each CI run report includes:
- **Duration**: Total execution time for each step and the pipeline overall.
- **Memory**: Peak memory usage in MB during execution.
- **Disk**: Storage consumed by output artifacts.
- **Success/Failure**: Status of each experimental step (US-1, US-2, US-3).

## Execution Context

Runs are executed with:
- **Hardware**: CPU-only environment (as per project constraints)
- **Python**: Standard CPython interpreter
- **Data**: Real measurements from simulated agent interactions (no fabricated data)

## Notes

- Game counts are reduced in CI mode for efficiency while maintaining statistical
 validity for demonstration purposes.
- All metrics are real measurements from actual code execution.
- Reports are timestamped for reproducibility.
