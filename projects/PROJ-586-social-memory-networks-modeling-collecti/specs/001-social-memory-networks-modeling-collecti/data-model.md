## Entity Definitions

### Agent
Single LLM instance participating in the multi-agent system.

| Field | Type | Description |
|-------|------|-------------|
| `agent_id` | str | Unique identifier (e.g., "agent_0", "agent_1") |
| `model_name` | str | HuggingFace model identifier (e.g., "distilgpt2") |
| `generated_text` | str | Text generated by the agent in a given turn |
| `memory_actions` | list[dict] | List of <MEMORY_ACTION> tokens executed by this agent |
| `memory_contributions` | list[str] | Facts contributed to the shared buffer by this agent |

### MemoryBuffer
Shared external memory store accessible by all agents.

| Field | Type | Description |
|-------|------|-------------|
| `key` | str | Unique identifier for the memory entry |
| `value` | str | Content of the memory entry (fact, cue, or response) |
| `timestamp` | float | Unix timestamp of creation |
| `owner_agent_id` | str | Agent ID that wrote this entry |

**Note**: The `access_count` field was removed from this entity as no plan phase tracks or uses it. Reserved for future forgetting/adaptation analysis.

### GameResult
Row in the output CSV summarizing one simulated game.

| Field | Type | Description | Constraints |
|-------|------|-------------|-------------|
| `game_id` | str | Unique game identifier | Unique |
| `specialization_index` | float | Distribution-based specialization metric | 0 â‰¤ value â‰¤ logâ‚‚(N_agents) |
| `retrieval_efficiency` | float | Proportion of successful retrievals vs. baseline | 0 â‰¤ value â‰¤ 1 |

| `agent_count` | int | Number of agents in this game | â‰Ą 2 |
| `token_threshold` | int | Context window size in tokens | 128, 256, or 512 |
| `game_success` | bool | Whether the game completed without errors | Boolean |

### MemoryAction
Structured token representing a read/write operation on the memory buffer.

| Field | Type | Description |
|-------|------|-------------|
| `action_type` | str | "READ" or "WRITE" |
| `key` | str | Memory key being accessed |
| `value` | str | Value being written (for WRITE) or query (for READ) |
| `agent_id` | str | Agent performing the action |
| `timestamp` | float | Unix timestamp |

### AnalysisOutput
Summary of statistical analysis results.

| Field | Type | Description |
|-------|------|-------------|
| `analysis_type` | str | "anova", "sensitivity", "power", "scaling" |
| `metric_name` | str | "specialization_index" or "retrieval_efficiency" |
| `statistic` | float | Test statistic (F-value, exponent, etc.) |
| `p_value` | float | P-value for hypothesis test |
| `corrected_alpha` | float | Bonferroni-corrected significance threshold |
| `significance_flag` | bool | True if p < corrected_alpha |
