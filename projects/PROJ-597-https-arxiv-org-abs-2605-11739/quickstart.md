# Quickstart Guide

Run the adapter to generate the analysis artifacts. This script is designed to finish in under 1 minute on a standard CPU.

```bash
python code/effopd_adapter.py
```

The script will output:
1.  Confirmation of synthetic data generation.
2.  Simulated analysis results for Module Allocation and Update Direction.
3.  Verification of EffOPD acceleration metrics.
4.  Real output files in `data/` and `figures/`.
