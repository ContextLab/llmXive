# Implementation Plan: Visual Complexity & Cognitive Load in Remote Meetings

## Summary

This project investigates the impact of visual complexity (entropy, variance, object count) on cognitive load (NASA-TLX, reaction time) during simulated remote meetings. The technical approach involves:
1.  **Stimulus Curation**: Use of **Curated Real-World Meeting Backgrounds** (verified, checksummed local archive of video frames) to ensure ecological validity.
2.  **Human Pilot**: A lightweight web interface (Streamlit) to collect human complexity ratings (n=20) to validate automated metrics.
3.  **Main Study**: A web-based experimental session presenting counterbalanced clips, collecting NASA-TLX and reaction-time data.
4.  **Statistical Analysis**: Linear Mixed-Effects Models (LMM) via `statsmodels`, with Benjamini-Hochberg correction, VIF checks, and sensitivity analysis sweeps.

All components are designed to run on a GitHub Actions free-tier runner (limited CPU and memory resources, no GPU).

> **Note on Dataset**: The project uses a **verified, checksummed local archive of real meeting background video frames** (derived from a public repository subset). No procedural generation or synthetic images are used. Real human data will be collected via the web app for the dependent variables (NASA-TLX/RT). **No fabricated or simulated data is used for the final hypothesis test.**

## Complexity Tracking

| Violation | Why Needed | Simpler Alternative Rejected Because |
|-----------|------------|-------------------------------------|
| **LMM over ANOVA** | Need to control for Participant ID as a random effect to handle repeated measures (multiple clips per participant). | Repeated measures ANOVA assumes sphericity and cannot easily handle missing data or random slopes. LMM is robust for this design. |
| **Real-World Stimuli** | Synthetic images lack ecological validity for "meeting background" context. | Procedural generation cannot replicate the complex, unstructured visual properties of real meeting environments. |
| **Sensitivity Analysis Sweep** | Required by FR-005/FR-005b to prove result stability across alpha thresholds. | A single p-value threshold (0.05) is insufficient to claim robustness in exploratory psychological research. |
| **No Fabricated Data** | Required by Constitution and Panel Concerns. | Synthetic data cannot answer the research question about cognitive load in real meetings. |
