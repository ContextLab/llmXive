# Methodology Documentation

## Overview
This project assesses the validity of p-values in high-dimensional data settings.

## Data Generation
Synthetic datasets are generated with controlled correlation structures and distributional violations.

## Analysis
P-value distributions are analyzed using Kolmogorov-Smirnov statistics and QQ-plots against a permutation-based Gold Standard.

## Reproducibility
All simulations use fixed seeds recorded in metadata files to ensure reproducibility.
