# PROJ-173: Quantifying the Impact of Transposable Element Activity on Gene Expression Variation in Drosophila

## Project Overview
This project implements a reproducible pipeline to analyze the impact of Transposable Element (TE) activity on gene expression variation using Drosophila data.

## Structure
- `code/`: Python source modules for data generation, preprocessing, association testing, and replication.
- `data/`: Input data, generated mock datasets, and analysis results.
- `tests/`: Unit and integration tests.
- `specs/`: Feature specifications and design documents.
- `docs/`: Documentation and reports.

## Quickstart
1. Install dependencies: `pip install -r requirements.txt`
2. Generate mock data: `python code/data_generator.py`
3. Run association analysis: `python code/association.py`
