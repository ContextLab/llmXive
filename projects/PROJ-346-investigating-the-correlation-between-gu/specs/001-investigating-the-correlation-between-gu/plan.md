# Implementation Plan: Investigating the Correlation Between Gut Microbiome Composition and Cognitive Flexibility

**Branch**: `001-gene-regulation` | **Date**: 2026-06-25 | **Spec**: `spec.md`
**Input**: Feature specification from `/specs/001-gene-regulation/spec.md`

## Summary

This project implements a computational pipeline to investigate the association between gut microbiome composition (16S rRNA data) and cognitive flexibility using publicly available datasets. The technical approach involves downloading and preprocessing microbiome and cognitive data, performing Spearman correlation analysis with Benjamini-Hochberg FDR correction, fitting regularized regression models (LASSO/Elastic Net) with demographic covariates, and conducting sensitivity analyses by age groups. 

**Critical Constraint**: The verified datasets (AGP, NHANES) do not contain paired individual-level data for both variables. Consequently, the primary analysis path (FR-003/FR-005) is conditional. If paired data is not found, the pipeline executes the **FR-008 Fallback**, which is defined here as generating a structured **Data Gap Report** rather than attempting invalid statistical synthesis. This ensures scientific integrity and compliance with the spec's requirement for a defined fallback path.

The pipeline is designed to run entirely on CPU within GitHub Actions free-tier constraints (limited cores, constrained RAM, bounded runtime).

## Critical Methodological Guardrails

1.  **NO Ecological Inference**: The pipeline will NOT compute correlations between independent cohorts (e.g., AGP vs NHANES) by aggregating to group means. This is scientifically invalid for individual-level questions.
2.  **NO Synthetic Linkage**: The pipeline will NOT create a synthetic dataset by matching on age/sex. This creates artifacts, not biological signals.
3.  **FR-008 Execution**: If individual-level data linkage fails (which is expected with AGP/NHANES), the pipeline MUST execute `07_gap_report.py` to generate a structured "Data Gap Report". This satisfies the spec's requirement for a fallback path without fabricating data.
4.  **Conditional Analysis**: Steps 03 (Correlation), 04 (Regression), and 05 (Sensitivity) are skipped if the data gap is detected. The pipeline transitions immediately to Step 07.
5.  **Success Criteria Handling**: If the data gap is detected, SC-001 (significant taxa) and SC-004 (robustness) are marked as "Not Measurable" in the final report, with a detailed explanation.
