# Processed Data Directory

This directory contains processed data artifacts generated by the llmXive pipeline.

## Embeddings

### `embeddings_{run_id}.parquet`

Generated by: `code/pipelines/run_baseline.py` (Task T015) and serialized by `code/embeddings/serializer.py` (Task T017)

**Schema:**
- `run_id`: str - Unique identifier for the embedding generation run
- `dataset_id`: str - Identifier of the source dataset
- `sample_id`: str - Unique identifier for the sample within the dataset
- `embedding`: list[float] - Flattened embedding vector (dimension varies by model)
- `modality`: str - Either 'image' or 'text'
- `model_name`: str - Name of the embedding model used (e.g., 'clip_vit_b_32', 'sentence_bert')
- `seed`: int - Random seed used for reproducibility (if applicable)
- `timestamp`: str - ISO format timestamp of when the embedding was generated

**Generation:**
- Embeddings are generated using CLIP ViT-B/32 for images and Sentence-BERT for text
- All operations run on CPU without gradient tracking
- Batch processing ensures memory safety
- Deterministic re-computation is guaranteed with `random_seed=42`

**Validation:**
- Each file includes a `run_id` to link embeddings to their generation parameters
- Files are validated for schema compliance before use in downstream tasks

## Metadata Statistics

### `metadata_stats_summary.csv`

Generated by: `code/analysis/metadata_stats.py` (Task T024)

**Schema:**
- `dataset_id`: str - Identifier of the dataset
- `cardinality`: int - Number of unique values per categorical feature
- `missingness`: float - Proportion of missing values
- `sparsity`: float - Proportion of zero values
- `variance`: float - Variance of numerical features

## Usage

To load embeddings:
```python
from embeddings.serializer import load_embeddings_from_parquet

embeddings = load_embeddings_from_parquet(
 'data/processed/embeddings_run_20240101_120000_abc12345.parquet',
 filter_by_dataset_id='my_dataset'
)
```
