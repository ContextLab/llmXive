# PROJ-071: Exploring the Correlation Between Molecular Complexity and Degradation Rates

## Overview
This project investigates the relationship between molecular complexity metrics (e.g., TPSA, Wiener Index) and pharmaceutical degradation rates using FDA-approved drug data.

## Setup
1. Create a virtual environment:
 ```bash
 python -m venv venv
 source venv/bin/activate # Linux/Mac
 # or
 venv\Scripts\activate # Windows
 ```
2. Install dependencies:
 ```bash
 pip install -r requirements.txt
 ```
