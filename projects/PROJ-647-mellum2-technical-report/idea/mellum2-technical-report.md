# Mellum2 Technical Report  

**Field**: linguistics  

## Research question  

How does syntactic complexity—measured by average dependency‑tree depth and branching factor—differ between spoken and written registers across typologically diverse languages?  

## Motivation  

Understanding register‑dependent syntactic variation informs theories of language processing, genre adaptation, and the design of NLP models that must handle both speech and text. Existing work has examined cross‑language syntactic differences, but systematic, quantitative comparisons of spoken versus written syntax across many languages remain scarce.  

## Related work  

- [Counting trees: A treebank‑driven exploration of syntactic variation in speech and writing across languages (2025)](https://arxiv.org/abs/2505.22774) — Introduces a treebank‑based framework for comparing syntactic structures between spoken and written registers; provides the primary methodological precedent.  
- [A Corpus‑based Study of Temporal Signals (2012)](https://arxiv.org/abs/1203.5066) — Shows how discourse‑level signals can be extracted from annotated corpora; demonstrates the feasibility of large‑scale corpus annotation for linguistic phenomena.  
- [Generating semantic maps through multidimensional scaling: linguistic applications and theory (2020)](https://arxiv.org/abs/2012.04946) — Applies multidimensional scaling to visualize linguistic dimensions; offers a technique for visualizing register‑based syntactic differences.  
- [Statistical methods for linguistic research: Foundational Ideas – Part II (2016)](https://arxiv.org/abs/1602.00245) — Reviews Bayesian and mixed‑effects modeling approaches for linguistic data; guides the choice of statistical analysis for cross‑language, cross‑register comparisons.  

## Expected results  

We anticipate finding systematic register effects: spoken language will exhibit shallower average tree depth and lower branching factor than written language, reflecting processing constraints. Confirmation will be a statistically significant register coefficient (p < 0.05) in a mixed‑effects regression controlling for language family; a null result (no significant difference) would challenge assumptions about register‑driven syntactic simplification.  

## Methodology sketch  

- **Data acquisition**  
  - Download Universal Dependencies (UD) treebanks for ≥15 typologically diverse languages (e.g., English, Mandarin, Arabic, Swahili) via their public URLs.  
  - For each language, extract sentences labelled with the `genre` metadata; retain those marked `spoken` (e.g., conversation, transcribed speech) and `written` (e.g., news, literature).  
- **Pre‑processing**  
  - Parse each sentence’s dependency tree (already provided in UD).  
  - Compute per‑sentence metrics: tree depth, mean branching factor, and number of nodes.  
- **Aggregation**  
  - Calculate register‑level averages per language (mean depth, mean branching).  
- **Statistical analysis**  
  - Fit a linear mixed‑effects model: syntactic metric ~ register + (1|language) + (1|language family).  
  - Use the Bayesian framework from *Statistical methods for linguistic research* to obtain credible intervals.  
  - Perform likelihood‑ratio tests to assess the significance of the register effect.  
- **Visualization**  
  - Apply multidimensional scaling (MDS) to the matrix of language‑by‑register syntactic metrics, following the approach of *Generating semantic maps*.  
  - Plot the resulting 2‑D semantic map, coloring points by register to illustrate clustering.  
- **Robustness checks**  
  - Repeat the analysis with alternative complexity measures (e.g., average dependency length).  
  - Conduct leave‑one‑language‑out cross‑validation to ensure results are not driven by a single language.
