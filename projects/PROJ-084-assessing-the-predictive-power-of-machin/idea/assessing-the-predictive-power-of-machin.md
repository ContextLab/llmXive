# Assessing the Predictive Power of Machine Learning for Organic Reaction Outcomes

**Field**: chemistry

Can machine learning models accurately predict the yield and selectivity of organic reactions based solely on reactant structures and reaction conditions? This project proposes to train and evaluate several readily available machine learning algorithms (e.g., random forests, support vector machines) on a publicly accessible dataset of organic reactions, such as the USPTO reactions dataset. The performance of these models will be assessed using metrics like R-squared and root mean squared error, with a focus on identifying structural features and reaction parameters that are most strongly correlated with reaction outcomes. This work could accelerate the discovery and optimization of chemical reactions by providing a computational screening tool for chemists.
