# Evaluating the Impact of Code Generation Models on Code Testability

## Project Overview

This project investigates how code generated by Large Language Models (LLMs) compares to human-written code in terms of testability, complexity, and functional correctness. We analyze code generated by models like `Salesforce/codegen-mono` and `CodeLlama` against the HumanEval benchmark.

## User Stories

### US1: Data Acquisition and Paired Analysis Pipeline
- Download HumanEval dataset with stratified sampling.
- Generate code using LLMs with retry logic.
- Compute metrics: Cyclomatic Complexity, Halstead Volume, Branch Coverage, Pass Rate.
- Produce a unified `metrics.json` dataset including sensitivity analysis data.

### US2: Statistical Comparison and Hypothesis Testing
- Perform Wilcoxon Signed-Rank tests on continuous metrics (Complexity, Halstead).
- Perform McNemar's test on binary pass rates.
- Perform Permutation tests on branch coverage.
- Conduct A Priori and Post-Hoc Power Analysis.

### US3: Visualization, Reporting, and Sensitivity Analysis
- Generate histograms and boxplots for all metrics.
- Compile a comprehensive Markdown report (`results_report.md`) with figures and statistical tables.
- Include sensitivity analysis comparing different model variants (e.g., CodeGen vs. CodeLlama).

## Prerequisites

- Python 3.8+
- pip
- Access to HuggingFace Hub (for datasets and models)

## Installation

1. Clone the repository.
2. Install dependencies:
 ```bash
 pip install -r requirements.txt
 ```
3. Ensure you have access to the required HuggingFace models (login if necessary):
 ```bash
 huggingface-cli login
 ```

## Usage

### Running the Pipeline

The main pipeline orchestrator executes the full workflow from data download to report generation:

```bash
python code/run_pipeline.py
```

### Individual Stages

- **Data Download**:
 ```bash
 python code/download_data.py
 ```
- **Code Generation**:
 ```bash
 python code/generate_code.py
 ```
- **Metric Analysis**:
 ```bash
 python code/analyze_metrics.py
 ```
- **Statistical Testing**:
 ```bash
 python code/statistical_tests.py
 ```
- **Report Generation**:
 ```bash
 python code/report_generator.py
 ```

## Output Artifacts

- `data/analysis/metrics.json`: Comprehensive metrics for all samples.
- `data/analysis/sensitivity_metrics.json`: Comparative analysis results.
- `results/figures/`: Generated plots (histograms, boxplots).
- `results_report.md`: Final Markdown report with all findings.
