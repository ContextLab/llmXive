## Research-question validation

### Phenomenon-vs-method check

**Verdict**: pass

The question asks about a relationship between neural signal dynamics (EEG complexity) and a cognitive state (fatigue), independent of any specific ML or computational method. The measurement tools (Lempel-Ziv, permutation entropy) are standard complexity metrics, not the question itself.

### Circularity check

**Verdict**: pass

The predictor (EEG complexity from resting-state EEG recordings) and the predicted variable (subjective fatigue from self-report questionnaires) come from independent measurement modalities. Neither is mechanically derived from the other, so any correlation would be empirically informative rather than by construction.

### Triviality check

**Verdict**: pass

Both outcomes are informative: a positive correlation would establish EEG complexity as a potential objective fatigue biomarker, while a null result would rule out this approach as viable. The literature gap analysis confirms this specific relationship hasn't been systematically tested, so the answer isn't predetermined by existing domain knowledge.

### Question-narrowing check

**Verdict**: pass

Names a domain relationship (EEG complexity → cognitive fatigue) rather than implementation constraints. The question asks about a neuroscientific phenomenon, not whether a specific method performs under specific resource limits.

### Overall verdict

**Verdict**: validated

All four checks pass without significant concerns. The research question is well-framed as a substantive scientific inquiry into a relationship between neural dynamics and cognitive state, with independent measurement modalities and informative outcomes in either direction.
