# Research Notes

## Data Sources
- Cochrane Library API

## Methodology
- Bootstrap resampling for subsampling
- Fixed Effects (FE) and Random Effects (RE) models
- DerSimonian-Laird (DL) for k >= 10
- REML for k < 10
- Generalized Additive Models (GAM) for threshold detection
- 1/sqrt(k) parametric model for validation
