# Ethics Documentation

This directory contains ethics-related artifacts for the study on visual detail and false memory.

## Contents

### Informed Consent Template (`informed_consent.md`)
A GDPR-compliant template for participant informed consent. Includes:
- Purpose of the study
- Procedures involved
- Potential risks and benefits
- Voluntary participation and right to withdraw
- Data handling and confidentiality
- Contact information for researchers

### IRB Placeholder (`irb_placeholder.md`)
A placeholder document for Institutional Review Board (IRB) approval. This should be replaced with the actual IRB approval letter once obtained. Includes:
- Study protocol summary
- Participant recruitment plan
- Data management plan
- Risk mitigation strategies
- IRB approval number and date

## Compliance
All ethics documentation adheres to GDPR requirements and institutional guidelines for human subjects research.
