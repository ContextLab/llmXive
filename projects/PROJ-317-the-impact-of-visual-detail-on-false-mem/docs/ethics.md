# Ethics Documentation

## Informed Consent Template

### Title: The Impact of Visual Detail on False Memory Susceptibility

### Purpose
This study investigates how visual detail in images influences the formation of false memories. Participants will view images, complete a distractor task, and answer recognition questions.

### Procedures
1. View a series of images (baseline and manipulated versions)
2. Complete a brief arithmetic distractor task
3. Answer recognition questions about image details (some true, some false)
4. Provide demographic information

### Risks
Minimal risk. Participants may experience mild fatigue from the distractor task. False memory tasks are non-invasive and pose no psychological harm.

### Benefits
Contribution to scientific understanding of memory formation and visual processing.

### Confidentiality
All data will be anonymized. Participant IDs will be used instead of names. Data will be stored on encrypted servers and accessible only to the research team.

### Voluntary Participation
Participation is entirely voluntary. Participants may withdraw at any time without penalty.
