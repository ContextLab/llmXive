# Evaluating the Impact of Variable Selection on Statistical Power in Linear Regression

## Project Structure

This project follows the standard research pipeline structure:

- `code/`: Source code for data loading, simulation, analysis, and visualization
- `data/raw/`: Raw data downloaded from external sources (OpenML)
- `data/processed/`: Processed simulation results and metrics
- `results/`: Generated plots, reports, and final outputs
- `tests/unit/`: Unit tests for individual components
- `tests/integration/`: Integration tests for end-to-end workflows

## Setup

1. Install dependencies:
 ```bash
 pip install -r requirements.txt
 ```

2. Run the pipeline:
 ```bash
 python code/pipeline.py
 ```
