## Research-question validation

### Phenomenon-vs-method check

**Verdict**: pass

The question asks about a substantive relationship between baseline network architecture and dynamic task-based communication in the brain. It is framed as a scientific question about brain organization and function, not about whether a specific method or algorithm performs well under constraints.

### Circularity check

**Verdict**: pass

The predictor (graph-theoretical metrics from resting-state functional connectivity) and the predicted variable (phase-locking value from task-based fMRI) are derived from different scanning sessions. While both use BOLD fMRI signals, resting-state and task-based acquisitions are temporally and contextually distinct, so the relationship is not mechanically guaranteed by construction.

### Triviality check

**Verdict**: pass

Both possible outcomes would be informative: a significant correlation would suggest trait-like network architecture constrains state-dependent communication, while a null result would indicate task-state synchrony is primarily state-dependent. Either finding addresses the mechanism underlying brain-cognition relationships.

### Question-narrowing check

**Verdict**: pass

The question names a domain relationship (resting-state topology → task-based synchrony) rather than an implementation constraint. It asks "how does X relate to Y" in the neuroscience domain, not "can method M achieve Z under budget B."

### Overall verdict

**Verdict**: validated

All four checks pass. The research question is properly framed as a substantive scientific inquiry about the relationship between baseline brain network organization and dynamic neural communication during cognition. The trait-state design (resting vs. task fMRI) avoids circularity, and either positive or null results would advance understanding of brain-cognition mechanisms.
