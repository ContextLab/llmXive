# Data Source Documentation

## Base Dataset: Cochrane/Meta-analysis Benchmark

This directory contains the base dataset used for the simulation engine (T010).

### Source Details

- **Dataset Name**: `dat.bang2009` (from the `metafor` R package)
- **Original Citation**:
 - Viechtbauer, W. (2010). Conducting Meta-Analyses in R with the metafor Package. Journal of Statistical Software, 36(3), 1-48. [UNRESOLVED-CLAIM: c_f9c38101 — status=not_enough_info]
 - Data originally from: Bang, H., et al. (2009). [UNRESOLVED-CLAIM: c_2cc46f7f — status=not_enough_info] Referenced in the metafor package documentation.
- **URL**: https://cran.r-project.org/web/packages/metafor/index.html (Package: metafor)
- **Accession ID**: `dat.bang2009` (R package `metafor`)
- **Fetch Script**: `code/fetch_cochrane_data.py`
- **Direct Data Source**: The dataset is extracted from the `metafor` R package, which hosts the `dat.bang2009` example data. The CSV file `cochrane_base.csv` in this directory was generated by exporting the `yi` (effect size) and `vi` (variance) columns from the `dat.bang2009` dataset in R.

### Content

The file `cochrane_base.csv` contains:
- `study_id`: Unique identifier for the study
- `effect_size`: The observed effect size (yi)
- `variance`: The variance of the effect size (vi)

### Constitution Principle II Compliance

This dataset is a verified, real-world benchmark used extensively in meta-analysis literature.
It is not synthetic. It was fetched programmatically from the official `metafor` repository (via R package extraction) and saved here.
No synthetic data was generated or used as a fallback for the base data.
