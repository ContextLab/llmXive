## Summary

This project implements a comparative analysis of Graph Neural Networks (GNNs) versus traditional machine learning baselines for predicting chemical reaction yields. The system ingests raw SMILES strings from the USPTO regression dataset, converts them into molecular graphs using RDKit, and trains a lightweight Message Passing Neural Network (MPNN) alongside a Random Forest baseline. The implementation strictly adheres to CPU-only constraints (GitHub Actions free-tier), ensuring the entire pipeline (data ingestion, 5-fold cross-validation, evaluation, and sensitivity analysis) completes within 10 hours. Key deliverables include a validated dataset of molecular graphs, trained models with uncertainty quantification (conformal prediction), and a statistical report on model performance and structural attribution.
