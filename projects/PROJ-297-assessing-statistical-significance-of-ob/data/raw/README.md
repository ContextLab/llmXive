# Raw Data Directory

This directory is intended for raw data downloaded from external sources (e.g., UCI, OpenML).

**Note**: Per project policy, data should be fetched programmatically at runtime using `code/loaders.py`.
This directory should generally be empty or contain only README files unless specific datasets are
pre-downloaded for offline CI environments.

If you see files here, they were likely generated by a previous run of `code/loaders.py` or
manually placed for debugging.
