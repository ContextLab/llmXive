# llmXive follow-up: extending "COLLEAGUE.SKILL: Automated AI Skill Generation via Expert Knowledge Di"

**Field**: computer science

## Research question

Does explicitly decoupling capability heuristics from behavioral style in LLM agent prompts reduce hallucinated expertise and style drift during long-horizon interactions compared to monolithic persona prompts?

## Motivation

Current persona systems often conflate tone with competence, causing agents to adopt a specific style while simultaneously hallucinating the associated expert knowledge. Isolating these tracks may allow smaller, CPU-tractable models to maintain consistent expertise without needing massive parameter counts to "memorize" a persona, addressing a critical bottleneck in deploying reliable agents on standard hardware.

## Related work

- [Enhancing Human-Like Responses in Large Language Models (2025)](https://arxiv.org/abs/2501.05032) — Provides a baseline for measuring "style consistency" by exploring techniques to enhance conversational coherence and natural language understanding in LLMs.
- [LLMORPH: Automated Metamorphic Testing of Large Language Models (2026)](https://arxiv.org/abs/2603.23611) — Offers a methodological precedent for the proposed rule-based scoring scripts by addressing the challenge of automated oracles for verifying output correctness without ground-truth labels.
- [Is Self-knowledge and Action Consistent or Not: Investigating Large Language Model's Personality (2024)](https://arxiv.org/abs/2402.14679) — Investigates the consistency between LLM self-reported traits and actual behavior, relevant to evaluating whether decoupled tracks improve the alignment between an agent's "stated" capability and its "performed" style.
- [Unmasking the Shadows of AI: Investigating Deceptive Capabilities in Large Language Models (2024)](https://arxiv.org/abs/2403.09676) — Examines deceptive behaviors in LLMs, providing a framework for identifying "hallucinated expertise" as a form of deceptive output that the proposed separation aims to mitigate.

## Expected results

We expect the "Separated Tracks" condition to yield significantly higher scores on heuristic adherence and lower rates of hallucination compared to the Monolithic condition, demonstrating that structural prompt separation compensates for limited model capacity. The evidence will be confirmed if statistical tests show a significant reduction in hallucination frequency (p < 0.05) while maintaining stylistic consistency metrics within a non-inferior margin of the monolithic baseline.

## Methodology sketch

- **Data Collection**: Download 50 distinct "expert profiles" (capability tracks, behavior tracks, and monolithic SKILL.md files) from the public COLLEAGUE.SKILL gallery (or a simulated public repository if the specific URL is unavailable, using standard GitHub raw content patterns). Generate 200 diverse, multi-turn task scenarios (e.g., code review, negotiation) using a rule-based Python script to ensure reproducibility and CPU-tractability.
- **Model Setup**: Load a quantized small language model (e.g., Llama-3-8B-Q4 or Phi-3-mini) using a CPU-only inference backend (e.g., `llama.cpp` or `transformers` with `torch_dtype=torch.float32` and no GPU) to simulate resource-constrained deployment.
- **Experimental Conditions**: For each profile-task pair, run three inference conditions: (1) Monolithic Prompt (original SKILL.md), (2) Separated Tracks (distinct system messages for capability and behavior), and (3) Generic Baseline.
- **Evaluation Metrics**: Implement a deterministic Python scoring script to evaluate outputs: (a) **Heuristic Adherence**: Binary match against capability track rules defined in the input profiles; (b) **Style Consistency**: Keyword/structure frequency analysis against behavior track definitions; (c) **Hallucination Rate**: Fact-checking against source traces using string matching and simple entailment checks against the ground-truth task context.
- **Statistical Analysis**: Aggregate scores across all 30,000 profile-task-condition combinations (50 profiles × 200 tasks × 3 conditions). Perform a repeated-measures ANOVA to test for significant differences in hallucination rates and heuristic adherence between the Monolithic and Separated Tracks conditions.
- **Independence Check**: Ensure the validation targets (hallucination rates, heuristic matches) are derived from independent source traces and rule definitions provided in the input profiles, not from the model's own generated text or the prompt structure itself. The "ground truth" for hallucination is the external task context, not the model's internal consistency.
