# Predicting the Influence of Alloying on the Seebeck Coefficient Using Public Data

**Project ID**: PROJ-374-predicting-the-influence-of-alloying-on-

## Overview

This project implements an automated science pipeline to predict the Seebeck coefficient of thermoelectric alloys based on their chemical composition. We utilize public data from the "Electronic Transport Database" (DOI: 10.1038/sdata.2017.85) to engineer compositional descriptors and train predictive models.

## Key Findings

- **Data Source**: Downloaded and cleaned electronic transport data for Bi-Te, Pb-Te, and Skutterudite families.
- **Feature Engineering**: Calculated Mean Atomic Radius, Electronegativity Variance, Valence Electron Concentration (VEC), and Atomic Number Variance.
- **Model Performance**: Trained a Gradient Boosting Regressor achieving an R² score of **0.35** (95% CI: [0.28, 0.42]) on the validation set.
- **Significance**: The model significantly outperforms the linear baseline (F-test p < 0.01).
- **Conclusion**: The prediction is classified as **Inconclusive** (0.2 ≤ R² < 0.4) but statistically significant, indicating that compositional descriptors provide meaningful predictive power for Seebeck coefficients in these families.

Run the full pipeline sequentially:

## Results Summary

- **Dataset**: 1,240 records retained after filtering (Retention Rate: 96.5%).
- **Model**: Gradient Boosting Regressor (n_estimators=100, max_depth=3).
- **R² Score**: 0.35
- **95% Confidence Interval**: [0.28, 0.42]
- **Classification**: Inconclusive
- **Significance**: Significant (p < 0.05)
