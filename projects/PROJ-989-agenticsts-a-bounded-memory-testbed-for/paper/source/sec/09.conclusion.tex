\section{Conclusion}
\label{sec:conclusion}

We present a long-horizon LLM-agent resource where memory is bounded,
typed, and ablatable at decision time. In the fixed-$A_0$ matrix the
largest observed difference appears when triggered $L_5$ skills are
enabled, though at this sample size it is directional rather than
statistically decisive; the $298$-trajectory archive supplies logs,
frozen stores, prompts, and scripts. By keeping game, denominators,
stores, and prompts aligned, the release lets future work ask whether
a new memory interface changes decisions rather than whether it
benefited from a different evaluation scaffold. Our evidence supports
a narrower conclusion: explicit, typed memory contracts make
long-horizon agent behavior easier to audit, reproduce, and ablate.
Whether a bounded contract outperforms a matched accumulating-context
design remains an open question for a controlled follow-up.
