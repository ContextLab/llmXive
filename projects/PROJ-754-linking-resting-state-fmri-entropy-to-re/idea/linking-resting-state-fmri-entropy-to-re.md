# Linking Resting‑State fMRI Entropy to Real‑World Decision Risk‑Taking  

**Field**: neuroscience  

## Research question  

Does individual variation in resting‑state fMRI entropy across cortical parcels predict differences in real‑world risk‑taking propensity?  

## Motivation  

Signal complexity (e.g., entropy) in spontaneous BOLD fluctuations is thought to index the brain’s capacity for flexible information processing, but empirical links to everyday decision behavior are sparse. Demonstrating a behavioral correlate would clarify the functional relevance of neural entropy and could inform neuro‑psychological models of risk‑taking.  

## Related work  

- [Fractal‑driven distortion of resting state functional networks in fMRI: a simulation study (2012)](https://arxiv.org/abs/1208.0924) — Shows that fractal‑like (scale‑invariant) dynamics, of which entropy is a quantitative proxy, can alter the topology of resting‑state networks, providing a theoretical basis for using entropy as a marker of neural complexity.  
- [Modeling and Interpreting Real‑world Human Risk Decision Making with Inverse Reinforcement Learning (2019)](https://arxiv.org/abs/1906.05803) — Introduces a computational framework for quantifying real‑world risk‑taking behavior, highlighting the importance of behavioral metrics (e.g., questionnaire scores) that can serve as independent outcome variables.  
- [Fractal‑based Correlation Analysis for Resting State Functional Connectivity of the Rat Brain in Functional MRI (2012)](https://arxiv.org/abs/1202.4751) – Demonstrates that fractal‑based analyses (including entropy) can capture meaningful functional connectivity patterns in animal rs‑fMRI, supporting the feasibility of applying similar entropy measures to human datasets.  

## Expected results  

We anticipate that higher multiscale sample entropy in frontoparietal and salience network parcels will be positively associated with higher scores on the Domain‑Specific Risk‑Taking Scale. Confirmation will be a set of parcel‑wise β‑coefficients that survive permutation‑based family‑wise error correction (p < 0.05). A null finding (no surviving parcels) would equally inform the field by delimiting the behavioral relevance of resting‑state entropy.  

## Methodology sketch  

- **Data acquisition**: Download the publicly released Human Connectome Project (HCP) 1200‑subject resting‑state fMRI and accompanying behavioral data (Domain‑Specific Risk‑Taking Scale) via the HCP S3 bucket.  
- **Preprocessing**: Use `fmriprep` (CPU‑only) to perform slice‑time correction, motion correction, spatial normalization to MNI space, and ICA‑FIX denoising; retain the minimally preprocessed 4 mm parcellated time series provided by HCP.  
- **Entropy computation**: For each parcel, compute multiscale sample entropy (m = 1–5) using the `pyEntropy` package; average across scales to obtain a single entropy value per parcel per subject.  
- **Covariate assembly**: Extract age, sex, and mean framewise displacement for each subject to include as nuisance regressors.  
- **Statistical modeling**: Fit a linear mixed‑effects model (parcel as random effect, subject as fixed effect) predicting risk‑taking score from parcel entropy, controlling for covariates.  
- **Permutation testing**: Generate 5 000 label‑shuffled permutations of the risk‑taking scores; recompute the model to build a null distribution of β‑values for each parcel.  
- **Multiple‑comparison correction**: Apply family‑wise error correction across parcels using the max‑statistic from the permutation null (threshold at p < 0.05).  
- **Robustness checks**: (a) Repeat analysis with alternative entropy parameters (different m, r); (b) Test whether results hold when using only the lowest‑motion participants (FD < 0.2 mm).  
- **Output**: Produce a parcel‑wise map of significant associations, summary statistics, and a brief report in PDF.
