# PROJ-185: Predicting Protein-Protein Interactions from Co-expression Networks

This project implements a pipeline to predict Protein-Protein Interactions (PPIs) in plants
using co-expression analysis of public transcriptomic data (GEO).

## Directory Structure

- `code/`: Source code for the pipeline (Python/R scripts)
- `tests/`: Unit and integration tests
- `data/`: Raw and processed data files (git-ignored)
- `results/`: Output artifacts (edge lists, metrics, reports)
- `docs/`: Documentation and design specs
- `contracts/`: JSON/YAML schemas for validation
- `specs/`: Project specifications and design documents
