# Data Schema: SDAR Results and Summary

This document defines the structure, data types, and units for the experimental output artifacts generated by the SDAR simulation (`code/sdar_sim.py`). These files correspond to the metrics described in `specs/001-https-arxiv-org-abs-2605-15155/spec.md`.

## 1. `data/sdar_results.csv`

Aggregated results across multiple noise levels and methods.

| Column Name | Data Type | Unit | Description | Mapping to Spec |
| :--- | :--- | :--- | :--- | :--- |
| `method` | String | N/A | The algorithm variant used (e.g., "SDAR", "Naive"). | `method` |
| `noise` | Float | 0.0–1.0 | The noise level injected into the teacher signal. | `noise` |
| `success_rate` | Float | 0.0–1.0 | Fraction of tasks completed successfully. | `success_rate` (User Story 3) |
| `avg_total_loss` | Float | N/A | Average total loss over the training steps. | `avg_total_loss` (derived from SDAR Gate Loss + RL Loss) |
| `avg_gate` | Float | 0.0–1.0 | Average gate activation ratio (proportion of teacher signals accepted). | `avg_gate` (Self-Distillation gating metric) |

## 2. `data/sdar_summary.json`

High-level summary of findings, including comparative analysis between SDAR and baseline methods.

### Root Structure
```json
{
  "paper": "string",
  "simulation_type": "string",
  "findings": [
    {
      "condition": "string",
      "sdar_success": number,
      "naive_success": number,
      "improvement": number,
      "sdar_avg_gate": number,
      "interpretation": "string"
    }
  ]
}
```

### Field Definitions

| Field | Data Type | Description | Mapping to Spec |
| :--- | :--- | :--- | :--- |
| `paper` | String | Title of the reproduced work. | N/A |
| `simulation_type` | String | Description of the environment (e.g., "Simplified Grid World"). | N/A |
| `findings[].condition` | String | The specific noise level or scenario being analyzed. | `noise` (context) |
| `findings[].sdar_success` | Float | Success rate for the SDAR method. | `success_rate` |
| `findings[].naive_success` | Float | Success rate for the Naive baseline. | `success_rate` (baseline) |
| `findings[].improvement` | Float | Difference: `sdar_success` - `naive_success`. | N/A |
| `findings[].sdar_avg_gate` | Float | Average gate activation for SDAR under this condition. | `avg_gate` |
| `findings[].interpretation` | String | Qualitative analysis of the results. | N/A |
