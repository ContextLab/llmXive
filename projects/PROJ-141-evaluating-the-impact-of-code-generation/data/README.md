# Data Directory

Store all datasets, CSV files, and raw data here.

## Subdirectories

- `raw/`: Raw downloaded data from external sources
- `processed/`: Processed and cleaned data
- `metadata.yaml`: Dataset metadata and version information
- `checksums.csv`: File checksums for integrity verification

## Important Files

- `metadata.yaml`: Contains dataset versions, commit hashes, timestamps
- `checksums.csv`: Contains checksums for all data files (Constitution Principle III)
