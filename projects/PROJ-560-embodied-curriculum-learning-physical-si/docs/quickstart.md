# Quick Start Guide: Embodied Curriculum Learning Pipeline

## Prerequisites

- Python 3.11+
- pip (package manager)

## Installation

1. Clone the repository and navigate to the project root.
2. Install dependencies:

```bash
pip install -r code/requirements.txt
```

## Usage

The pipeline is executed via the CLI entry point `code/src/cli.py`.

### Mode 1: Synthetic Data Generation (Validation)

Use this mode to generate a deterministic dataset for testing the pipeline
or when public data is missing required fields.

```bash
python code/src/cli.py --mode=synthetic --seed=42
```

**Outputs**:
- `data/synthetic/generated_data.csv`: The synthetic dataset.
- `data/synthetic/mapping_log.json`: Documentation of the physics-to-math mapping.
- `data/processed/results.json`: Statistical analysis results.

### Mode 2: Secondary Analysis (Public Data)

Use this mode to analyze an existing public dataset.

```bash
python code/src/cli.py --mode=secondary_analysis --input=data/raw/your_dataset.csv --seed=42
```

**Automatic Fallback**: If the input CSV lacks the `instruction_type` column,
the system will automatically switch to synthetic generation to ensure valid
analysis can proceed.

**Outputs**:
- `data/processed/results.json`: Full statistical report.
- `data/derivation_logs/skipped_records.log`: Log of excluded rows.

### Sensitivity Analysis

To run a sensitivity sweep over inclusion thresholds:

```bash
python code/src/cli.py --mode=secondary_analysis --input=data/raw/your_dataset.csv --sweep_thresholds=0.01,0.05,0.10
```

**Note**: If N < 30, the sweep will be skipped with a warning.

## Understanding the Results

The output file `data/processed/results.json` contains:
- `t_statistic`: The calculated t-value.
- `p_value`: The raw p-value.
- `bonferroni_p_value`: The p-value after correction.
- `effect_size`: Cohen's d.
- `associational_framing`: A boolean flag and text confirming the non-causal
 interpretation of the results.
- `power`: The achieved statistical power.
- `robustness_warning`: Present if sensitivity analysis was skipped or unstable.

## Important: Associational Framing

**All results generated by this tool are strictly associational.**
Do not interpret the statistical association as proof of causation. The system
is designed to avoid causal language in its automated outputs to maintain
scientific rigor.
