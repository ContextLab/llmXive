# Qwen-VLA Cross-Embodiment Transfer Study

Research pipeline for investigating cross-embodiment transfer in Vision-Language-Action models using Qwen-VL.

## Project Structure

- `src/`: Core implementation code
- `tests/`: Unit and contract tests
- `data/`: Datasets, checkpoints, and evaluation results
- `scripts/`: Auxiliary scripts and batch runners

## Setup

1. Create a virtual environment:
 ```bash
 python -m venv.venv
 source.venv/bin/activate
 ```

2. Install dependencies:
 ```bash
 pip install -e ".[dev]"
 ```
