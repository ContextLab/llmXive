# Polymer Degradation Pathways Prediction

This project implements a pipeline for predicting polymer degradation pathways using Graph Neural Networks.

## Setup

```bash
python code/setup_project.py
```

## Usage

```bash
# Run ingestion
python code/ingest.py

# Run preprocessing
python code/preprocess.py

# Train model
python code/train.py
```

## Data

- `data/raw/`: Raw downloaded data
- `data/processed/`: Processed datasets
- `data/reports/`: Analysis reports
