# Implementation Plan: Bayesian Nonparametrics for Anomaly Detection in Time Series

**Branch**: `001-bayesian-nonparametrics-anomaly-detection` | **Date**: 2026-06-26
**Spec**: `specs/001-bayesian-nonparametrics-anomaly-detection/spec.md`

## Summary

This feature implements a research pipeline to evaluate Gaussian Process (GP) regression with Sparse Variational Inference (VI) against traditional Statistical Process Control (SPC) methods and a lightweight Variational Auto-Encoder (VAE) for anomaly detection in univariate time series. The system ingests public time series data, injects synthetic anomalies with known ground truth, runs inference within strict CPU/memory constraints (limited RAM, 6h runtime), and produces statistical comparisons (F1-scores, Bootstrap Confidence Intervals) and sensitivity analyses.

## Complexity Tracking

| Violation | Why Needed | Simpler Alternative Rejected Because |
|-----------|------------|-------------------------------------|
| **Sparse Variational Inference** | Full GP is computationally expensive (cubic complexity) and will OOM on 7GB RAM for typical time series lengths. | Dense GP kernel is computationally intractable for the required window sizes within the 6h limit. |
| **Synthetic Anomaly Injection** | Real-world labeled anomaly data is scarce and inconsistent; synthetic injection allows controlled hypothesis testing. | Using only public anomaly datasets would limit the ability to test specific shift magnitudes (FR-011) and types (FR-009). |
| **Multiple Baselines (SPC + VAE)** | To establish relative advantage, comparison against both statistical (Shewhart/CUSUM) and learning-based (VAE) methods is required. | Comparing only to one baseline would not validate the specific claim of Bayesian nonparametric superiority across different anomaly types. |
| **Bootstrap CI over Wilcoxon** | F1-scores are bounded and skewed; small sample sizes make Wilcoxon low-power. | Wilcoxon tests median differences which may be misleading for bounded metrics; Bootstrap provides robust confidence intervals for the difference in means. |
