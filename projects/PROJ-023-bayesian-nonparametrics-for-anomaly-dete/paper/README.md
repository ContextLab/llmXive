# Research Pipeline Documentation

## Overview
This directory contains the documentation, figures, and final reports for the
**Bayesian Nonparametrics for Anomaly Detection in Time Series** project.

## Structure

- `README.md`: This file.
- `figures/`: Generated plots and visualizations.
 - `fig1_timeseries.png`: Time series with injected anomalies and detection scores (T028).
 - `fig2_method_comparison.png`: Method comparison (F1 vs. shift magnitude) and correlation matrices (T029).
- `results.md`: Comprehensive summary of findings, statistical test results (p-values),
 and associational claims (T030).

## Pipeline Version
This documentation corresponds to pipeline version `1.0.0` as defined in `data/VERSION.txt`.

## Key Artifacts

### Figures
All figures are generated by scripts in `code/scripts/`:
- `render_fig1.py` generates the time series visualization.
- `render_fig2.py` generates the comparative analysis plots.

### Results
The `results.md` file aggregates:
- Precision, Recall, F1, and AUC-ROC metrics.
- Bootstrap Confidence Intervals (Bonferroni corrected).
- Wilcoxon signed-rank test or paired t-test results comparing Bayesian vs. Baseline methods.
- Correlation analysis between performance and anomaly shift characteristics.

## Reproducibility
To regenerate all figures and results:
1. Ensure the environment is set up per `requirements.txt`.
2. Run the full pipeline:
 ```bash
 python code/scripts/download_data.py
 python code/scripts/inject_anomalies.py
 python code/scripts/bayesian_gp.py
 python code/scripts/baseline_shewhart.py
 python code/scripts/baseline_cusum.py
 python code/scripts/baseline_vae.py
 python code/scripts/evaluate.py
 python code/scripts/render_fig1.py
 python code/scripts/render_fig2.py
 ```
3. Review `data/results/` and `paper/figures/` for outputs.
