# Quickstart: Evaluating Statistical Validity of Public A/B Test Summaries  

This guide walks you through a fresh clone of the repository to the point where you have a completed audit, dashboard, and reproducibility manifest.

## 1. Clone the Repository
```bash
git clone https://github.com/your-org/ab-test-audit.git
cd ab-test-audit
```

## 2. Set Up a Python Virtual Environment
```bash
python3 -v venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```
