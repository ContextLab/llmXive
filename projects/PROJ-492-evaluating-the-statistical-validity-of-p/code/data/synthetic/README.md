# Synthetic Validation Dataset (T026)

This directory contains the synthetic dataset generated by task T026 for evaluating
the inconsistency detection component (T029).

## Files

- `synthetic_summaries.json`: Generated A/B test summaries with ground truth labels.
- `ground_truth_labels.json`: Explicit mapping of record IDs to inconsistency status.

## Generation

The dataset is generated using `code/src/audit/synthetic.py` with a fixed random seed
(SEED=42) to ensure reproducibility. It contains at least 10,000 records covering
both binary and continuous outcome types.

## Usage

To regenerate the dataset:

```bash
python code/src/audit/synthetic.py
```

To evaluate the inconsistency detection component:

```bash
python code/src/audit/evaluation.py
```

## Expected Metrics

The evaluation (T029) asserts:
- Precision >= 90%
- Recall >= 80%

If these thresholds are not met, the script exits with error code 1 and logs `ERR-800`.
