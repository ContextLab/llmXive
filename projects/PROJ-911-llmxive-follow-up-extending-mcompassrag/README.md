# PROJ-911: GraphCompass - Topological Predictors of Semantic Coherence
#
# This project implements a CPU-constrained RAG evaluation pipeline using
# graph topology to predict semantic coherence.
#
# ## Structure
# - `code/`: Python implementation modules
# - `data/`: Raw, processed, and result artifacts
# - `tests/`: Unit and integration tests
# - `specs/`: Design documents and user stories
# - `contracts/`: Schema definitions for artifacts
#
# ## Prerequisites
# Python 3.11+
#
# ## Setup
# 1. Install dependencies: `pip install -r requirements.txt`
# 2. Run data setup: `python code/setup_data_dirs.py`
# 3. Run pipeline: `python code/run_pipeline.py`
