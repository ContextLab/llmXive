## Unresolved concerns

- FR-007 requires validating task metadata against a defined `constraint_keywords` list in the template schema. T009 creates `template_map.json` (ID to filename) and T014 implements `validator.py`, but no task exists to define, populate, or load the `constraint_keywords` list itself. Without this data, the conflict detection logic in FR-007 is unimplementable.
- FR-003 mandates executing 7 specific autonomous agents with a concurrency limit of 7 and a 24-hour wall-clock budget. T022 implements an agent loader factory, and T023 implements the execution loop, but no task explicitly defines the configuration or instantiation of the 7 specific agents from the original study, nor is there a task to implement the specific 24-hour budget enforcement logic or concurrency controller required by FR-003.
