# Statistical Analysis of Publicly Available Chess Game Data

This project analyzes publicly available chess game data (from Lichess) to predict Elo ratings and understand the impact of various game features.

## Prerequisites

- Python 3.11+
- pip

## Installation

```bash
pip install -r requirements.txt
```

## Usage

See `quickstart.md` for detailed instructions.

## Project Structure

- `code/src/`: Source code
- `code/tests/`: Test suites
- `code/data/`: Data storage (raw, processed, results)
- `code/specs/`: Design documents and contracts
- `code/config.py`: Configuration constants
