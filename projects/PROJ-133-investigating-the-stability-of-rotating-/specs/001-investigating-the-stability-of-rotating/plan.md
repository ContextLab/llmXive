# Implementation Plan: Investigating the Stability of Rotating Bose-Einstein Condensates with Dipolar Interactions

**Branch**: `001-gene-regulation` | **Date**: 2026-10-27 | **Spec**: `specs/001-gene-regulation/spec.md`
**Input**: Feature specification from `/specs/001-gene-regulation/spec.md`

## Summary

This project implements a CPU-tractable numerical solver for the time-dependent Gross-Pitaevskii (GPE) equation with dipolar interactions to generate stability phase diagrams for rotating Bose-Einstein Condensates (BECs). The system simulates a grid of feasible dimensions (for full grid scan feasibility) across a defined parameter space (rotation frequency Ω, dipolar strength ε_dd, particle number N), detects vortices via phase-winding analysis, calculates stability metrics (vortex density, radial variance, structure factor), and performs statistical aggregation (Two-Way ANOVA, Dunnett's post-hoc) over multiple noise-seed repeats per configuration. The output is a validated phase map distinguishing stable, metastable, and unstable regimes, generated entirely within the constraints of a GitHub Actions free-tier runner (CPU-only, ≤7GB RAM, ≤6h runtime).

> **CRITICAL SPEC-PLAN ALIGNMENT GAP**: The source specification (FR-005, SC-003) mandates a "one-sample t-test" and "one-way ANOVA" using "vortex-number retention fraction." Scientific review (Panel Concerns) confirms this design is statistically invalid for multi-factor grid analysis and fails when initial vortex count is zero. This plan implements the scientifically valid **Two-Way ANOVA** and **Vortex Density** metric. The implementation will proceed with the corrected methodology, but the Spec **MUST** be amended to reflect Two-Way ANOVA and Vortex Density before the project can be considered "research_complete" to satisfy FR-005/SC-003.
