# PhysBrain 1.0 Technical Report

**Field**: computer science / robot learning

## Research question

How do human-video-derived physical priors affect downstream robot manipulation performance when controlling for architectural differences between learning systems?

## Motivation

Current vision-language-action models for robotics rely heavily on robot trajectory data, which provides limited coverage of diverse physical interactions. Human video data offers a complementary signal for learning physical understanding at scale. However, prior work has not isolated the contribution of human priors from architectural confounds, leaving uncertainty about whether observed gains stem from the data modality or model design choices.

## Literature gap analysis

### What we searched

Queried arXiv abs/2605.15298 (PhysBrain 1.0) and one additional paper on Physics-Informed Neural Networks. Total returned: 2 papers, only one directly addressing the research question.

### What is known

- [PhysBrain 1.0 Technical Report (2026)](https://arxiv.org/abs/2605.15298) — Establishes that incorporating human video data alongside robot trajectories can improve success rates on robot manipulation benchmarks (LIBERO, SimplerEnv), but does not isolate human priors from architectural confounds.

### What is NOT known

No published work has systematically ablated the contribution of human-video priors from dual-pathway architectures in robot learning. The marginal gain attributable specifically to human priors versus architectural changes remains unquantified. No analysis exists of whether human priors transfer across different robot embodiments without retraining.

### Why this gap matters

Robotics labs deploying VLA models need to know whether collecting human video data provides measurable return on investment versus investing in architectural improvements. Clarifying this would enable more efficient data collection strategies and prevent misattribution of performance gains in follow-on work.

### How this project addresses the gap

This project re-analyzes the PhysBrain 1.0 benchmark results using public data to conduct ablation studies that isolate human priors from architectural effects. By comparing single-pathway and dual-pathway models trained on equivalent robot-only versus robot-plus-human data, we quantify the marginal contribution of each factor.

## Expected results

We expect to find that human-video priors contribute measurable but modest gains (5-15% improvement) on physical manipulation benchmarks, with architectural changes accounting for a larger portion of observed performance differences. Null results (no significant difference between priors conditions) would suggest that robot trajectory data alone suffices for the evaluated tasks, challenging the value proposition of human-video data collection.

## Methodology sketch

- Download PhysBrain 1.0 benchmark datasets from Hugging Face (physbrain-bench, roboset) and OpenML robot manipulation collections
- Extract human-video priors and robot trajectory data splits from the public PhysBrain 1.0 repository (github.com/contextlab/physbrain)
- Implement single-pathway and dual-pathway VLA architectures using lightweight PyTorch models (≤100M parameters for GHA memory constraints)
- Train models on three conditions: (1) robot-only data, (2) human-only data, (3) combined data with matched sample sizes
- Evaluate success rates on LIBERO and SimplerEnv benchmark tasks using the standard evaluation scripts from the original repositories
- Apply paired t-tests across 10 random seeds to compare performance differences between conditions (alpha=0.05, report 95% confidence intervals)
- Conduct ablation by freezing pathway weights and measuring marginal contribution of each data modality to final task success
