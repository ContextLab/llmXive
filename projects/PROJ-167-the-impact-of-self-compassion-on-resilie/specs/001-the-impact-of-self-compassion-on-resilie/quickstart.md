# Quickstart: The Impact of Self‑Compassion on Resilience to Negative Feedback

## Prerequisites

- Python 3.10 or higher
- `pip` package manager
- Access to the OSF dataset (publicly available)

## Installation

1.  **Clone the repository** (or navigate to the project directory).
2.  **Create a virtual environment**:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```
3.  **Install dependencies**:
    ```bash
    pip install pandas numpy statsmodels seaborn matplotlib requests jinja2 pyyaml
    ```
    *(Note: `requirements.txt` will be generated by the Implementer Agent with pinned versions.)*

## Execution

Run the full pipeline:

```bash
python code/main.py
```

### Expected Flow

1.  **Download**: Fetches data from OSF, computes SHA-256, saves to `data/raw/`.
2.  **Clean**: Validates columns, drops missing rows, encodes variables.
3.  **Model**: Fits ANCOVA models for Anxiety, Rumination, and Self-Efficacy.
4.  **Robustness**: Computes VIF, Bootstrap CIs, and Homogeneity tests.
5.  **Visualize**: Generates simple slope plots.
6.  **Report**: Generates `reports/report.html`.

## Output Verification

- Check `data/raw/` for the downloaded CSV and a `.sha256` file.
- Check `reports/` for `report.html`.
- Check `code/` for generated PNG files (e.g., `anxiety_simple_slopes.png`).
- If the dataset is missing required columns, the script will exit with code 1 and a specific error message.

## Troubleshooting

- **Error: `DATA_UNAVAILABLE`**: The dataset at the OSF URL does not contain the required columns. Verify the OSF project contents.
- **Error: `POWER_INSUFFICIENT`**: N < 92. The analysis will proceed, but the report will contain a warning.
- **Error: `BOOTSTRAP_CONVERGENCE_CAVEAT`**: Bootstrap did not stabilize within 10,000 resamples. Results are still reported but with caution.
