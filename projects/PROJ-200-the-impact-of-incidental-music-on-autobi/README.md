# llmXive Project - The Impact of Incidental Music on Autobiographical Memory Retrieval

## Overview

This project investigates the impact of incidental music on autobiographical memory retrieval. It involves data ingestion, processing, statistical modeling, and analysis to understand how music can trigger or enhance memories.

## Data Sources

* MSD (Million Song Dataset)
* AMT (Amazon Mechanical Turk) data

## Key Components

* **Data Ingestion:** Downloading and filtering MSD and AMT datasets.
* **Cue Matching:** Matching free-text memory cues to MSD tracks.
* **Statistical Modeling:** Fitting linear mixed-effects models to analyze the relationship between music exposure and memory vividness/valence.
