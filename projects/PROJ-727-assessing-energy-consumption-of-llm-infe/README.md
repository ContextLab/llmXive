# PROJ-727: Assessing Energy Consumption of LLM Inference for Code Completion

This project evaluates the energy efficiency of different LLM architectures (GPT-2, CodeBERT, StarCoder-1B) when performing code completion tasks on the HumanEval dataset.

## Structure
- `data/`: Raw and processed datasets
- `src/`: Source code for inference, evaluation, and analysis
- `docs/`: Documentation and reports
- `results/`: Generated visualizations and statistical reports

## Prerequisites
- Python 3.10+
- See `requirements.txt` for dependencies.
