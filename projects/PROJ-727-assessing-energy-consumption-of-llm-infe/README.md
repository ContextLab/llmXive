# PROJ-727: Assessing Energy Consumption of LLM Inference for Code Completion

This project investigates the energy cost of running LLMs for code completion tasks on CPU hardware.

## Structure
- `code/`: Source modules for inference, evaluation, analysis, and visualization
- `data/`: Raw and processed datasets
- `specs/`: Feature specifications and design documents
- `tests/`: Unit and integration tests

See `quickstart.md` in the project root for setup and execution instructions.
