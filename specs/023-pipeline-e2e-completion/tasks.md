## Implementation Strategy

MVP = Phase 1 + 2 + US1 (T001–T009): with only that, papers start exiting
review and scheduler capacity frees. Then US2 (funnel) lands the second P1.
US4 and US5 land the automation + gate while US3's traversal progresses in
wall-clock time. US6 and Polish close out. Commit after each task or
logical group; the standing FR-012 rule (encountered defects fixed
generally + regression-tested in the same effort) applies to every task.
