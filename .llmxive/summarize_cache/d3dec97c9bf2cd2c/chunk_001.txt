**Note**: Tasks marked Sequential cannot run in parallel with their dependencies. Group C tasks run in parallel ONLY after all Phase 2 dependencies complete. T001 is NOT marked [P] as it depends on D003-D006; it runs in parallel with T002-T005 only after Phase 2 schemas are complete.
